## 🖥️ 通用操作 (General)
- `Ctrl+Shift+P`, `F1`：显示命令面板
- `Ctrl+P`：快速打开，跳转到文件
- `Ctrl+Shift+N`：新建窗口/实例
- `Ctrl+Shift+W`：关闭窗口/实例
- `Ctrl+,`：打开用户设置
- `Ctrl+K Ctrl+S`：打开键盘快捷键设置
---

## ✏️ 基础编辑 (Basic Editing)
- `Ctrl+X`：剪切当前行（未选中文本时）
- `Ctrl+C`：复制当前行（未选中文本时）
- `Alt+↑ / ↓`：上下移动当前行
- `Shift+Alt+↑ / ↓`：复制当前行到上/下方
- `Ctrl+Shift+K`：删除当前行
- `Ctrl+Enter`：在下方插入新行
- `Ctrl+Shift+Enter`：在上方插入新行
- `Ctrl+Shift+\`：跳转到匹配的括号
- `Ctrl+] / [`：缩进 / 反缩进当前行
- `Home / End`：移动到行首 / 行尾
- `Ctrl+Home`：跳转到文件开头
- `Ctrl+End`：跳转到文件末尾
- `Ctrl+↑ / ↓`：滚动一行
- `Alt+PgUp / PgDn`：滚动一页
- `Ctrl+Shift+[`：折叠当前区域
- `Ctrl+Shift+]`：展开当前区域
- `Ctrl+K Ctrl+[`：折叠所有子区域
- `Ctrl+K Ctrl+]`：展开所有子区域
- `Ctrl+K Ctrl+0`：折叠所有区域
- `Ctrl+K Ctrl+J`：展开所有区域
- `Ctrl+K Ctrl+C`：添加行注释
- `Ctrl+K Ctrl+U`：移除行注释
- `Ctrl+/`：切换行注释
- `Shift+Alt+A`：切换块注释
- `Alt+Z`：切换自动换行
---

## 📍 导航 (Navigation)
- `Ctrl+T`：显示所有符号
- `Ctrl+G`：跳转到指定行
- `Ctrl+P`：跳转到文件
- `Ctrl+Shift+O`：跳转到符号（函数、变量等）
- `Ctrl+Shift+M`：显示问题（错误/警告）面板
- `F8`：跳转到下一个错误或警告
- `Shift+F8`：跳转到上一个错误或警告
- `Ctrl+Shift+Tab`：在编辑器历史中切换
- `Alt+← / →`：后退 / 前进（导航历史）
- `Ctrl+M`：切换 Tab 是否移动焦点
---

## 🔍 查找与替换 (Search and Replace)
- `Ctrl+F`：查找
- `Ctrl+H`：替换
- `F3 / Shift+F3`：查找下一个 / 上一个
- `Alt+Enter`：选择所有匹配项
- `Ctrl+D`：添加下一个匹配项到选择中
- `Ctrl+K Ctrl+D`：将上一个选中的文本添加为下一个匹配项
- `Alt+C / R / W`：切换大小写敏感 / 正则模式 / 全字匹配
---

## ✨ 多光标与选择 (Multi-Cursor and Selection)
- `Alt+点击`：插入多个光标
- `Ctrl+Alt+↑ / ↓`：向上 / 向下插入光标
- `Ctrl+U`：撤销上一个光标操作
- `Shift+Alt+I`：在选中每一行的末尾插入光标
- `Ctrl+L`：选择当前整行
- `Ctrl+Shift+L`：选择所有匹配的文本
- `Ctrl+F2`：选择所有当前单词的匹配项
- `Shift+Alt+→ / ←`：扩大 / 缩小选择范围
- `Shift+Alt + 拖动鼠标`：列（框选）选择
- `Ctrl+Shift+Alt`（配合拖动鼠标）列（框选）选择，可以用于一列中多行同时修改
---

## 📦 **列（框选）选择**
- `(箭头键)` + `Ctrl+Shift+Alt`：列（框）选择
- `Ctrl+Shift+Alt + PgUp/PgDn`：列选择时整页向上 / 向下移动光标
---

## 🧠 **智能提示与代码辅助（Rich languages editing）**
- `Ctrl+Space` 或 `Ctrl+I`：触发智能提示（补全）
- `Ctrl+Shift+Space`：触发参数提示（如函数参数）
- `Shift+Alt+F`：格式化整个文档
- `Ctrl+K Ctrl+F`：格式化选中区域
- `F12`：跳转到定义
- `Alt+F12`：预览定义（嵌入窗口）
- `Ctrl+K F12`：在旁侧打开定义
- `Ctrl+.`：快速修复（比如导入未引用的包）
- `Shift+F12`：查看所有引用
- `F2`：重命名符号
- `Ctrl+K Ctrl+X`：删除行尾空格
- `Ctrl+K M`：更改文件语言（比如从纯文本改为 Python）
---

## 🧩 **编辑器管理（Editor Management）**
- `Ctrl+F4` / `Ctrl+W`：关闭当前编辑器
- `Ctrl+K F`：关闭文件夹
- `Ctrl+\`：分屏显示编辑器
- `Ctrl+1 / 2 / 3`：聚焦第 1/2/3 编辑器组
- `Ctrl+K Ctrl+← / →`：切换聚焦到上一个 / 下一个编辑器组
- `Ctrl+Shift+PgUp / PgDn`：将编辑器向左 / 向右移动
- `Ctrl+K ← / →`：移动当前活动编辑器组
---

## 📁 **文件管理（File Management）**
- `Ctrl+N`：新建文件
- `Ctrl+O`：打开文件
- `Ctrl+S`：保存文件
- `Ctrl+Shift+S`：另存为
- `Ctrl+K S`：全部保存
- `Ctrl+F4`：关闭当前文件
- `Ctrl+K Ctrl+W`：关闭所有文件
- `Ctrl+Shift+T`：重新打开上一个关闭的文件
- `Ctrl+K Enter`：保持预览模式的编辑器打开
- `Ctrl+Tab`：切换到下一个打开文件
- `Ctrl+Shift+Tab`：切换到上一个打开文件
- `Ctrl+K P`：复制当前文件路径
- `Ctrl+K R`：在资源管理器中显示当前文件
- `Ctrl+K O`：在新窗口中打开当前文件
---

## 🖥️ **界面显示（Display）**
- `F11`：切换全屏
- `Shift+Alt+0`：切换编辑器布局（横向 / 纵向）
- `Ctrl+= / -`：放大 / 缩小界面
- `Ctrl+B`：切换侧边栏显示
- `Ctrl+Shift+E`：显示资源管理器 / 聚焦
- `Ctrl+Shift+F`：显示全局搜索
- `Ctrl+Shift+G`：显示源代码管理
- `Ctrl+Shift+D`：显示调试面板
- `Ctrl+Shift+X`：显示扩展插件面板
- `Ctrl+Shift+H`：在文件中替换
- `Ctrl+Shift+J`：切换搜索选项详细信息
- `Ctrl+Shift+U`：显示输出面板
- `Ctrl+Shift+V`：打开 Markdown 预览
- `Ctrl+K V`：在旁边打开 Markdown 预览
- `Ctrl+K Z`：Zen 模式（进入沉浸式模式，退出：按两次 Esc）
---

## 🐞 **调试（Debug）**
- `F9`：设置 / 取消断点
- `F5`：开始 / 继续调试
- `Shift+F5`：停止调试
- `F11 / Shift+F11`：单步进入 / 单步跳出
- `F10`：单步跳过
- `Ctrl+K Ctrl+I`：显示悬停提示
---

## 🖥️ **集成终端（Integrated Terminal）**
- `Ctrl+` (一般是 `Ctrl+``) 打开集成终端
- `Ctrl+Shift+` (一般是 `Ctrl+Shift+``) 新建终端
- `Ctrl+C`：复制终端选中内容
- `Ctrl+V`：粘贴到活动终端
- `Ctrl+↑ / ↓`：滚动终端一行
- `Shift+PgUp / PgDn`：滚动终端一页
- `Ctrl+Home / End`：滚动到终端顶部 / 底部
