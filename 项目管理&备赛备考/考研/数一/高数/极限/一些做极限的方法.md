Q：何时使用泰勒何时使用洛必达？
A：当一个式子比较**简单**且为**未定式** $\frac {0}{0}$、$\frac {∞}{∞}$ 时，使用洛必达，其余使用泰勒
#### 夹逼准则
> `A ≤ f(x) ≤ A` -> `f(x) = A`
1. $h(x) ≤ f(x) ≤ g(x)$ 
2. $\lim h(x)=A,\lim g(x)=A$ 
=> $\lim f(x)$ 存在，$f(x) = A$ 
---
#### 洛必达法则
###### 前提：
1. 极限式：$\frac{0}{0}$​ 或 $\frac{\infty}{\infty}​$ 
2. 分子与分母在该点附近 **可导** 
3. 分母的 **导数** 在该点附近 **不为 0** 
4. 分子分母求导后的 **极限存在**（或为 ±∞±∞±∞）
###### 结论：
$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

#### 拉格朗日中值定理
###### 前提：
1. $f(x)$ 在闭区间 `[a,b]`上连续
2. $f(x)$ 在开区间 `(a,b)` 内可导
###### 结论：
则存在至少一个点 $c \in (a,b)$ ，使得：
$$f'(c)=\frac{f(b)-f(a)}{b-a} 、{f(b)-f(a)} = f'(c)·(b-a)$$
​%% 即函数在某一点的瞬时变化率 = 整段区间的平均变化率 %%

#### 泰勒公式
###### 前提：
1. $f(x)$ 在闭区间 $[a, b]$ 上具有直至 $n$ 阶的**连续导数**
2. $f(x)$ 在开区间 $(a, b)$ 内具有 **$n+1$ 阶导数**
3. $x, x_0$ 为区间内两个不同的点
###### 结论：
则在 $x$ 与 $x_0$ 之间至少存在一个点 $\xi$，使得：
$$f(x) = f(x_0) + f'(x_0)(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \dots + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)$$
**对于 $x_0$ 的取值：**
- 通常使用临近的点，如不知道 $\sqrt{4.1}$ 是多少，但知道 $\sqrt{4} = 2$ 
- 常会用到等价无穷小的替换
###### 其中余项（Lagrange Remainder）：
$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}$$
