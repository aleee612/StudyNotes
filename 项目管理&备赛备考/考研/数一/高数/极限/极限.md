### 函数极限
#### 性质
##### 唯一性
1. 存在的**充要**条件： 左右极限存在且都相等 -> $\displaystyle \lim _{x \to x_{0}}f(x)=A=\lim _{x \to x_{0}^-}f(x)=\lim _{x \to x_{0}^+}f(x)$ 
2. 关于 ∞ ，需分为 +∞ 和 -∞ 讨论；当 +∞ 和 -∞ 均存在极限时，认为$\displaystyle \lim _{x \to ∞}f(x)$ 不存在（唯一性）
3. 在分段函数中，需注意分段两侧的表达式是否相同
##### 局部有界性
> if ( $\displaystyle \lim _{x \to x_{0}}f(x)=A$  ){                  当极限存在
> 	number M, δ；                 存在任意M, δ
> 	if ( 0 < |$x-x_{0}$| < δ ){        当$x-x_{0}$在0~δ中
> 		|$f(x)$| ≤ M;                $f(x)$ 的绝对值小于M
> 	}
> }
1. 极限存在 => 函数有界（不可逆推）
2. $y=f(x)$ 在 `[a,b]` 上连续，则说明该函数在此区间有界
3. $y=f(x)$ 在 `(a,b)` 上连续 + $\displaystyle \lim _{x \to b^-}f(x)\lim _{x \to a^+}f(x)$ 都存在（此处见唯一性）=> 在此区间（开）有界
4. 有界函数与有界函数的 和，差，积 仍为有界函数（但商不一定）-> 也就是初等函数的和差积都为有界
##### 局部保号性
> $\displaystyle \lim _{x \to x_{0}}f>0$     ->    f > 0
> $\displaystyle \lim _{x \to x_{0}}f< 0$     ->    f < 0
> f ≥ 0             ->    $\displaystyle \lim _{x \to x_{0}}f$ ≥ 0
> f ≤ 0             ->    $\displaystyle \lim _{x \to x_{0}}f$ ≤ 0
---
#### 无穷小
###### 定义： 若
$$\lim_{x\to 0}\frac{f(x)}{g(x)}=1$$
则称 $f(x)$ 与 $g(x)$ 是 **等价无穷小**，记作 $f(x)\sim g(x)$（ 当$x\to 0$ ）
- =0 -> 高阶无穷小
- =$\infty$ -> 低阶无穷小
###### 常用等价：
$$x \sim \sin x \sim \tan x\sim \ln(1+x)\sim e^x-1$$
$$x-\ln(1+x)\sim 1-\cos x \sim \frac{x^2}{2}  $$
$$(1+x)^\alpha-1\sim \alpha x（\alpha为常数）$$
###### 性质：
- 乘除可直接替换：若 $f\sim g$，则 $\frac{f}{h}\sim\frac{g}{h}$、$fh\sim gh$ 
- 加减一般不能直接替换（$sinx−x\not\sim 0$）

### 数列极限
关于[[数列]]
#### 性质
##### 唯一性
与函数极限相同
##### 全局有界性
- 函数的有界性为局部，而数列强调全局有界
- 对数列中任意一项都有$|a_n|≤M$ 
##### 保号性
> $\displaystyle \lim _{n \to \infty}x_n=a>b$     ->   存在 N>0，当 n>N 时，有$x_n>b$ 
> $\displaystyle \lim _{n \to \infty}x_n=a<b$     ->   存在 N>0，当 n>N 时，有$x_n<b$ 
> $x_n≥b$ & $\displaystyle \lim _{n \to \infty}x_n=a$      ->      $a≥b$ 
> $x_n≤b$ & $\displaystyle \lim _{n \to \infty}x_n=a$      ->      $a≤b$ 
数列极限没有强调局部保号性，但也强调存在正整数N，当n>N时才有$a_n>0$或$a_n<0$
- 它的意义在于： 数列前 n 项是什么样（是正、是负、还是忽正忽负）统统不重要
- 只要极限 $A > 0$，那么总能找到一个“分水岭” $N$。在这个 $N$ 之后的无穷多项，必须乖乖地全是正数m

---
#### 有界
定义：对所有正整数n，存在正实数M，有$|a_n|≤M$ 
> [!note] 几种方法证明有界
> 1. 找M
> 2. 放缩法
> 	1. $if： u_i\geq 0\Rightarrow1·u_{max} \leq u_1+u_2+···+u_n\leq n·u_{max}$ 
> 	2. $else： n·u_{min} \leq u_1+u_2+···+u_n\leq n·u_{max}$ 
> 3. 找最值
> 4. [[不等式|基本不等式]]
##### 单调有界准则 => 证极限存在
> 单增+上界 => 极限存在
> 单减+下界 => 极限存在
**证明单调：**
1. 作差法：$x_{n+1}-x_n\geq0$ ， $x_{n+1}-x_n\leq0$  
2. 等比法：$\frac{x_{n+1}}{x_n}\geq1$  ，$\frac{x_{n+1}}{x_n}\leq1$  
3. 数学归纳法：验$x_1$ -> 设$x_k$ -> 证$x_{k+1}$ 
4. 不等式
5. 压缩映射原理：
	1. $\left| x_{n+1}-a\right| \leq k\left| x_{n}-a\right|$ **=>** $\{x_n\}$ 收敛于 $a$ 
	2. $x_{n+1}=f\left( x\right)$ **+** $a$ 是$f(x)=x$的唯一解 **+** $\left| f'\left( x\right) \right| \leq k <1$  **=>** $\{x_n\}$ 收敛于 $a$
