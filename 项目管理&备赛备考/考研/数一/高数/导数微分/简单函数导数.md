### 基本初等函数
| **函数 f(x)**     | **导数 f′(x)**        | **备注**                       |
| --------------- | ------------------- | ---------------------------- |
| 常数 $C$          | $0$                 | 永远的零                         |
| 幂函数 $x^\mu$     | $\mu x^{\mu-1}$     | $\mu$ 为常数                    |
| 指数函数 $a^x$      | $a^x \ln a$         | 特别地：$(e^x)' = e^x$           |
| 对数函数 $\log_a x$ | $\frac{1}{x \ln a}$ | 特别地：$(\ln x)' = \frac{1}{x}$ |
| 正弦 $\sin x$     | $\cos x$            |                              |
| 余弦 $\cos x$     | $-\sin x$           | **注意负号**                     |
| 正切 $\tan x$     | $\sec^2 x$          | $\sec^2 x = 1 + \tan^2 x$    |
| 余切 $\cot x$     | $-\csc^2 x$         |                              |
| 正割 $\sec x$     | $\sec x \tan x$     |                              |
| 余割 $\csc x$     | $-\csc x \cot x$    |                              |

### 反三角函数求导
- $(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$
- $(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$
- $(\arctan x)' = \frac{1}{1+x^2}$
- $(\text{arccot } x)' = -\frac{1}{1+x^2}$

---
### 双曲函数与反双曲函数
**双曲正弦：** $(\sinh x)' = \cosh x$
**双曲余弦：** $(\cosh x)' = \sinh x$ （注意：这里没有负号！）
**双曲正切：** $(\tanh x)' = \frac{1}{\cosh^2 x} = \text{sech}^2 x$

### 导数的运算法则
**四则运算：**
- $(u \pm v)' = u' \pm v'$
- $(uv)' = u'v + uv'$ （前导后不导 + 前不导后导）
- $(\frac{u}{v})' = \frac{u'v - uv'}{v^2}$ （上导下不导 - 上不导下导，除以下方的平方）
**复合函数（链式法则）：**
若 $y = f(u)$ 且 $u = g(x)$，则 $\frac{dy}{dx} = f'(u) \cdot g'(x)$
**反函数求导：**
    $\frac{dy}{dx} = \frac{1}{\frac{dx}{dy}}$ （即：反函数的导数等于原函数导数的倒数）

### 高阶导数：莱布尼茨公式
$$(uv)^{(n)} = \sum_{k=0}^{n} C_n^k u^{(n-k)} v^{(k)}$$
> 这个公式长得跟二项式展开极其相似，把幂次换成导数阶数即可

---
>[!note] 注意点
>1. **关于负号：** 凡是以 "c" 开头的三角函数（$\cos, \cot, \csc$），其求导结果都带负号
>2. **熟记特殊值：** 比如 $(\ln|x|)' = \frac{1}{x}$，无论 $x$ 正负，这个公式都成立，在做不定积分时非常有用
>3. **勤练复合求导：** 考研不会考你简单的 $\sin x$，通常是 $\sin(e^{x^2})$ 这种“洋葱皮”式的函数，要学会由外向内一层层剥开
