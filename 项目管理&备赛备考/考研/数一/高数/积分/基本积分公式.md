#### 基本积分公式
| 函数类型  | 被积函数                                              | 不定积分                                                           |
| ----- | :------------------------------------------------ | :------------------------------------------------------------- |
| 简单函数  | $$0$$                                             | $$C$$                                                          |
|       | $$k（常数）$$                                         | $$kx+C$$                                                       |
|       | $$x^n,(n\neq-1)$$                                 | $$\displaystyle\frac{x^{n+1}}{n+1}+C$$                         |
|       | $$\displaystyle\frac1x$$                          | $$\ln\|x\|+C$$                                                 |
|       | $$a^x,(a>0,a\neq1)$$                              | $$\displaystyle\frac{a^x}{\ln a}+C$$                           |
| e相关   | $$e^x$$                                           | $$e^x+C$$                                                      |
| 三角函数  | $$\sin x$$                                        | $$-\cos x+C$$                                                  |
|       | $$\cos x$$                                        | $$\sin x+C$$                                                   |
|       | $$\tan x$$                                        | $$-\ln\|\cos x\|+C$$                                           |
|       | $$\cot x$$                                        | $$\ln\|\sin x\|+C$$                                            |
|       | $$\sec x$$                                        | $$\displaystyle\ln\|\sec x+\tan x\|+C$$                        |
|       | $$\csc x$$                                        | $$\displaystyle\ln\|\csc x-\cot x\|+C$$                        |
| 平方    | $$\sin^2x$$                                       | $$\frac{x}{2}-\frac{\sin2x}{4}+C$$                             |
|       | $$\cos^2x$$                                       | $$\frac{x}{2}+\frac{\sin2x}{4}+C$$                             |
|       | $$\tan^2x$$                                       | $$\tan x-x+C$$                                                 |
|       | $$\cot^2x$$                                       | $$-x-\cot x+C$$                                                |
|       | $$\sec^2x$$                                       | $$\tan x+C$$                                                   |
|       | $$\csc^2x$$                                       | $$-\cot x+C$$                                                  |
| 相乘    | $$\sec x\tan x$$                                  | $$\sec x+C$$                                                   |
|       | $$\csc x\cot x$$                                  | $$-\csc x+C$$                                                  |
| 反三角函数 | $$\displaystyle\frac1{1+x^2}$$                    | $$\arctan x+C$$                                                |
|       | $$\displaystyle\frac1{\sqrt{1-x^2}}$$             | $$\arcsin x+C$$                                                |
|       | $$\displaystyle\frac1{a^2+x^2},(a>0)$$            | $$\displaystyle\frac1a\arctan\frac xa+C$$                      |
|       | $$\displaystyle\frac1{\sqrt{a^2-x^2}},(a>0)$$     | $$\arcsin\frac xa+C$$                                          |
| ln函数  | $$\ln x$$                                         | $$x\ln x-x+C$$                                                 |
|       | $$\displaystyle\frac1{x^2-a^2}$$                  | $$\displaystyle\frac1{2a}\ln\left\|\frac{x-a}{x+a}\right\|+C$$ |
|       | $$\displaystyle\frac1{a^2-x^2}$$                  | $$\displaystyle\frac1{2a}\ln\left\|\frac{a+x}{a-x}\right\|+C$$ |
|       | $$\displaystyle\frac1{\sqrt{x^2+a^2}}$$           | $$\displaystyle\ln\left\|x+\sqrt{x^2+a^2}\right\|+C$$          |
|       | $$\displaystyle\frac1{\sqrt{x^2-a^2}},(x>\|a\|)$$ | $$\displaystyle\ln\left\|x+\sqrt{x^2-a^2}\right\|+C$$          |

#### 常用换元型积分
| 函数类型   | 被积函数                                             | 积分结果                                      |
| ------ | :----------------------------------------------- | :---------------------------------------- |
| e相关    | $$f'(x)e^{f(x)}$$                                | $$e^{f(x)}+C$$                            |
| 幂但非e   | $$f'(x)a^{f(x)}$$                                | $$\displaystyle\frac{a^{f(x)}}{\ln a}+C$$ |
| ln逆    | $$\displaystyle\frac{f'(x)}{f(x)},(f(x)\neq0)$$  | $$\ln\|f(x)\|+C$$                         |
| 三角函数相关 | $$f'(x)\sin f(x)$$                               | $$-\cos f(x)+C$$                          |
|        | $$f'(x)\cos f(x)$$                               | $$\sin f(x)+C$$                           |
| 反三角    | $$\displaystyle\frac{f'(x)}{1+[f(x)]^2}$$        | $$\arctan(f(x))+C$$                       |
|        | $$\displaystyle\frac{f'(x)}{\sqrt{1-[f(x)]^2}}$$ | $$\arcsin(f(x))+C$$                       |
