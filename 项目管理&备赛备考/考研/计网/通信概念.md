#### 信道
- 信道：物理媒介，如电缆，无线电波
- 信号 ：电信号、光信号或无线电波
- 信源：产生/发送信息
- 信宿：接收信息
#### 码元
- 即一个脉冲信号
- 有可能携带数据：1bit，2bit，4bit
**调制技术** ：
- 作用：改变码元携带数据大小
- 方式：设置**模拟信号**中信号的频率、相位、振幅 等
- 详见：[[编码和调制#调制方法]] 
**振幅** ：
- 四种，低（00）、中（01）、高（10）、很高（11）
- 改变的是传输的数据
- 发一个脉冲信号，它的振幅是低，那就说明发送的是 00（也就是 2bit），它的振幅是中（01），发送的就是 01（也就是 2bit）… 以此类推
#### 波特率&比特率
**波特率**： 指每**秒**钟传输多少个**码元**
**比特率：** 指每**秒**钟传输多少个**比特
- 一个码元能携带 1bit 数据，那么 **比特率** = **波特率**。
- 一个码元能携带 2bit 数据，那么 **比特率** = 2 倍的 **波特率**。
- 一个码元能携带 4bit 数据，那么 **比特率** = 4 倍的 **波特率**。
- 即：**波特率** M 对应的 **比特率** 为 Mn bit/s
#### 速率（比特率）
- 连接到计算机网络上的节点在数字信道上传送数据的速率
- 别名：数据传输速率、数据率、比特率
- 单位： b/s（比特／秒）or bit/s（ bps )
- $kb/s（k = 10^3 ）、Mb/s（M = 10^6 ）、 Gb/s（G = 10^9）$ 
#### 带宽
- 最高数据传输**速率**
通信领域：
- 指通信线路能够通过的信号频率范围
- 单位：赫兹（Hz）
计算机网络：
- 数字信道所能达到的 **最大数据传输速率**
- 单位： b/s（比特／秒）
与吞吐量：
- **带宽**：理论上的最大传输速率，就像是一条高速公路的最大车道数。
- **吞吐量**：实际传输速率，受到网络拥塞、协议开销、设备性能等因素影响，往往低于带宽。

### 奈奎斯特定理
![[Pasted image 20260119220846.png]]
**核心思想**：
- 即使在没有噪声的理想信道中，信道的带宽仍然限制了其数据传输能力
- 为了避免相邻码元之间的干扰（即码间串扰），每秒钟最多只能传输有限数量的码元，这个数量与带宽直接相关
> _在理想无噪声信道中，**极限传输速率取决于带宽和码元的离散电平数 。**_
### 香农定理
![[Pasted image 20260119221522.png]]
**核心思想：**
- 即使信道中存在噪声
- 只要信息传输速率低于某个极限值（即信道容量），就一定可以通过某种编码方式实现几乎无差错的传输。
- 这个极限值取决于信道的带宽和 **信噪比**。
> 1. 带宽 ( B ) 越大，信道容量 ( C ) 越大：这意味着更宽的信道可以传输更多的数据。
> 2. **信噪比** ( NS​ ) 越大，信道容量 ( C ) 越大：这意味着信号相对于噪声越强，传输速率就越高。
> 3. S​/N 是 **线性比值，不是分贝！**
### 错题
> 下列因素中，不会影响信道数据传输速率的是（ ）
> 	A.信噪比      B.频率宽带      C.调制速度      D.信号传播速度
- 所谓影响信道数据传输速率，指的是信道容量，即带宽和码元的离散电平数
- 传播速度快 → 延迟短  
	码元发得快 → 数据多  
	数据传输速率 = 码元速率 × 每码元信息量，和传播速度无关
