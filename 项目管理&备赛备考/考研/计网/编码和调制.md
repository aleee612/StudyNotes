### 调制方法
**数字信号** -> **模拟信号**
![[Pasted image 20260206224619.png]]

| **调制技术** | 改变载波的 | 表示方法例子                | 常见应用                 |
| -------- | ----- | --------------------- | -------------------- |
| **ASK**  | 幅度    | _0: 无信号；1: 最大幅度的信号_   | 光纤通信                 |
| **FSK**  | 频率    | _0: 900 Hz；1: 901 Hz_ | 低速无线通信、电话系统          |
| **PSK**  | 相位    | _0: 0°相位；1: 180°相位_   | 高速无线通信，例如 Wi-Fi      |
| **QAM**  | 幅度和相位 | _16 种不同的幅度和相位组合_      | 数字电视、Wi-Fi、有线和无线通信系统 |
