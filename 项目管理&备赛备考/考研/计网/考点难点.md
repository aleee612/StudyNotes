> [!warning]+ 做题须知
> 区分两种 MB：
> 
> **计算机存储容量的 MB（二进制）**：
   1 MB = $2^{20}$ Byte = 1,048,576 Byte
   常用于 **内存、ROM、显存、硬盘理论容量**（尤其在计组和 MAR 题里）
   >
   **数据传输速率的 MB（十进制）**：
   1 MB = $10^{6}$ Byte = 1,000,000 Byte
   常用于 **带宽 / 硬盘标称速度 / 网络速度**
   Mbps、MB/s 里往往默认十进制（工程计量单位）

### 数据链路层
#### 主要功能
1. 帧定界（定义数据格式）
2. 差错检测&纠错
3. 流量控制（不是拥塞控制，拥塞是防止过多分组注入网络导致网络性能下降，是网络层的事）
4. 传输透明（主要体现在交换机）
5. 可靠传输
