### 概念
1. **主频**：中央处理器（**CPU**）的 时钟频率（Clock Frequency），表示 **CPU** 每秒可以执行多少个基本 **时钟周期**
	- 单位： **MHz**， **GHz**，它决定了 **时钟振荡** 的快慢：
		- **1 Hz**：每秒 1 次 **时钟振荡**
		- **1 MHz** = 10^6 次/秒 = 每秒经历 百万个 时钟周期
		- **1 GHz** = 10^9 次/秒 = 每秒经历 10亿个 时钟周期 = 1纳秒ns 一个时钟周期 
![[Pasted image 20260112160309.png]]
- **时钟周期** 是硬件层面的最小时间单位，**时钟脉冲信号** 的宽度，即连续两个 **脉冲** 之间的时间间隔，称为 **时钟周期**。
	- 不同的设备可能使用不同的时钟周期
- **机器周期** 是由多个 **时钟周期** 组成，用于完成一个基本操作（如取指令、存储数据）
- **指令周期**（**CPI**） 是由多个 **机器周期** 组成，用于完成一条指令的执行
    - 也就是一条指令所需要的平均 **时钟周期**
- 三者具有层级关系：**时钟周期** < **机器周期** < **指令周期**
1. **IPC**：也称 **每周期指令**，即每个 **时钟周期** 执行的平均指令数，是 **CPI** 的 _倒数_
2. **IPS**：也称 **每秒指令数**，即每秒平均能够执行多少条指令
	 $IPS=主频*IPC$ 
3. **MIPS**：也称 **每秒百万指令数**，即每秒能够执行 **多少个** 百万的指令
4. **FLOPS**（每秒浮点运算次数）
	衡量：计算机浮点运算性能
	评估对象： **GPU** 或 **AI 芯片** 的性能
	每次指标提升： ×10^3 
	- **MFLOPS**（Million FLOPS）: 百万（ 10^6 ）次每秒
	- **GFLOPS**（Giga FLOPS）: 十亿次（ 10^9 ）每秒
	- **TFLOPS**（Tera FLOPS）: 万亿次（ 10^12 ）每秒
	- **PFLOPS**（Peta FLOPS）: 千万亿次（ 10^15 ）每秒
	- **EFLOPS**（Exa FLOPS）: 百亿亿次（ 10^18 ）每秒
	- **ZFLOPS**（Zetta FLOPS）: 万亿亿次（ 10^21 ）每秒
5. **字长**：CPU一次操作中能处理的二进制数字位数
6. **数据通路带宽**： **CPU** 和 **内存** 之间传输数据的宽度，通常以 **位** 为单位表示
	- 比如，**64 位** 的数据总线意味着每个 **时钟周期** 内，可以传输 **64 位** 的数据。
7. **主存储器**：
	- **MAR** 的 **位数** 反映了 **可寻址范围** 的最大值（不一定是存储器的实际容量），与 **地址总线** 的 **位数** 一致。
	- 这里主要看清题目中的 **按字节寻址** 和 **按字寻址**
	- 内存空间 / 寻址单位 = 可寻址范围，通过 可寻址范围 计算 **MAR** 的 **位数**。
	- 比如主存地址空间大小为 128KB= 2^17 B，字长16 位（2B），按字寻址，则 MAR 位数为16。
8. **吞吐量**：
	- **单位时间** 内，系统可以完成的 **任务数量**。
	- 在计算机网络中： **数据的传输速率**；
	- 在 **计组** 中：**每秒执行的操作数**。
### 错题
> 程序 P 在机器 M 上的执行时间是 20 秒，编译优化后，P 执行的指令数减少到原来的 70%，而 CPI 增加到原来的 1.2 倍，则 P 在 M 上的执行时间是（ ）。
	A.8.4 秒    B.11.7 秒    C.14 秒    D.16.8 秒
- $T=20*0.7*1.2=16.8$ 
- CPI作为指令周期，其增加是会导致时间的延长的

> 下列关于 CPI 和 CPU 时钟周期的叙述中，错误的是（ ）。
	A.不同类型指令的 CPI 可能不一样
	B.程序的 CPI 与 Cache 缺失率无关
	C.单周期 CPU 的时钟周期以最耗时指令所用的时间为准
	D.流水线 CPU 的时钟周期以最长流水段所用时间为准
- 程序的 CPI 与 Cache 缺失率有关，因为 Cache 缺失会导致额外的内存访问延迟，从而增加访存型指令比如 `LOAD` 执行的平均时钟周期数
- 单周期CPU指的是每条指令在一个时钟周期内完成，因此时钟周期需要以最耗时的为准
- 而在流水线 CPU 中，时钟周期必须能够满足流水线中**所有阶段**的时间要求，因此以最长为准
