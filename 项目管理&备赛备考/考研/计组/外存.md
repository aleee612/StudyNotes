### 机械硬盘 HDD
| 概念               | 作用                    | 关键特点                       |
| ---------------- | --------------------- | -------------------------- |
| **Head<br>磁头**   | 位于读写臂上<br>读取和写入盘片表面数据 | 每个盘片表面通常对应一个磁头；磁头移动来访问不同磁道 |
| **Track<br>磁道**  | 存放一圈连续的数据             | 数据沿圆环排列，每个盘面有很多磁道          |
| **Cylinder（柱面）** | 同半径磁道集合               | 用于定位数据位置（CHS寻址的一部分）        |
| **Sector（扇区）**   | 最小存储单位                | 常见大小：512B 或 4KB（新硬盘多为4KB）  |
#### CHS 地址
- **柱面号**
    $柱面号位数=⌈log_2​（柱面数量）⌉$ 
- **磁头号**：目标扇区在哪一个盘面上  
    $磁头号位数=⌈log_2​（盘面数量）⌉$ 
- **扇区号**：目标扇区具体是磁道中的哪一个扇区  
    $扇区号位数=⌈log_2​（磁道中的扇区数量）⌉$ 
_也就是全部 $log_2$ 后向上取整_
#### 性能指标
$$平均存取时间=寻道时间+旋转延迟时间+传输时间$$
##### 数据传输率
$$数据传输率 = 磁盘转速 * 每条磁道容量$$
### RAID（非重点）
- **磁盘镜像**
- **条带化**
- **奇偶校验**
- **Cache 机制**
### 固态硬盘  SSD
| 特点             | 内容                                   |
| -------------- | ------------------------------------ |
| **无机械部件**      | 采用闪存芯片存储数据，无盘片、磁头、机械臂等机械部件，可靠性高，抗震性强 |
| **读写速度快**      | 无寻道时间和旋转延迟，随机读写性能高                   |
| **访问时间低**      | 延迟通常为微秒级，远低于 HDD 的毫秒级                |
| **寿命较长（整体设备）** | 采用 ECC、磨损均衡等技术延长使用寿命                 |
| **无碎片性能问题**    | 访问速度与 数据物理位置 基本无关                    |
| **容量成本高**      | 大容量 SSD（如 4TB+）价格明显高于 HDD            |
| **写入次数有限**     | 闪存单元有擦写寿命（P/E Cycle），高强度写入场景需关注      |
