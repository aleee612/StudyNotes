### 加法运算电路
#### 半加器
![[Pasted image 20260122170338.png]]
- 可见A,B分别为一个加数一个被加数
- S作为最终**结果**，使用了一个或门
- C表示是否**进位**，使用一个与门
- 处理位数：两位
缺点：
- 无法表示和计算由低位来的进位输入
#### 全加器
![[Pasted image 20260122171939.png]]
- 半加器pro版

#### 加法器
![[Pasted image 20260122212456.png]]
- 一个 n 位的 **加法器** 可以通过串联 n 个 **全加器** 来实现
- **ZF**：当 **A+B** 的结果中的每一位都为 0 时，**ZF** 为 **1**，所以 ZF=!(F0​F1​⋯Fn−1​) 。
- **SF**：**A+B** 的结果的正负取决于输出结果的最高位，所以 SF=Fn−1​ 。
- **CF**：
    - 对于 **加法**，进位即最高位， CF=Cout​
    - 对于 **减法**， CF=¬Cout​
- **OF**： OF=Cn−1​⊕Cout​ ，（⊕：异或）
    - 如果 Cn−1​ 和 Cout​ 不同，则表示 **符号位** 的变化导致溢出，即：
        - 一个正数加上另一个正数，结果是负数。
        - 一个负数加上另一个负数，结果是正数。
    - 如果 **Cin** 和 **Cout** 相同，则没有溢出发生。
