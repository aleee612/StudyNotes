1. **寄存器** （Registers）:
    - 位置：处理器内部
    - 最快速存储单元，最小存储空间
2. **一，二，三级缓存** （L1，L2，L3 Cache）:
    - 位置：处理器内部
    - 比 **寄存器** 空间稍大，逐级递减，缓存空间逐级递增
    - 分为数据缓存（用于存储数据）和指令缓存（用于存储指令）
3. **主存** （RAM）:
    - 即内存，比缓存慢，但比硬盘快得多，并且容量比缓存大
    - 存储对象：正在运行的**程序**和当前使用的**数据**
4. **辅助存储** :
    - 提供大量的永久存储（**仅**辅存可永久性存储数据）
    - 常见设备：机械硬盘（HDD）和固态硬盘（SSD）
- 上一层的存储器作为低一层存储器的高速缓存
### 半导体随机存储器 RAM
随机：随机访问
- 可以以几乎恒定的时间访问存储器中的任意位置
- 与其他存储技术（如磁盘或磁带）不同，后者需要按顺序访问数据
分类：SRAM 与 DRAM
####  静态随机存取存储器 SRAM（static）
- 使用触发器（通常由 4-6 个晶体管组成）构成存储器的基本存储单元![[Pasted image 20260130185303.png]]
- 通过将触发器组织成二维阵列构成存储器，每行连接一个字线（Word Line），控制该行所有单元的访问
- 每列连接一对位线（Bit Line），用于传输数据
**稳定性：**
- 只要电源持续供应，数据无需刷新即可保持稳定
**主要运用：**
- CPU 缓存（L1/L2/L3 Cache）
- 寄存器
- 嵌入式系统中的小型高速存储
**宏观：**![[Pasted image 20260130193926.png]]
#### 动态随机存储器 DRAM（Dynamic）
- 使用 1 个晶体管和 1 个电容器（1T1C 结构）构成存储器的基本存储单元，存储每个比特的数据
- 电容器存储电荷，表示 1 比特数据（有电荷为 1，无电荷为 0）。
- 晶体管作为开关，控制电容器与外部位线（bit line）的连接，读写数据。![[Pasted image 20260130190216.png]]
**稳定性：**
- 电容器具有 **动态特性**， 会随时间漏电，导致数据丢失
- 因此需要 **定期刷新**（通常每几毫秒）以恢复电荷
**特点：**
- 低成本
- 高容量
**主要用于**：
- 计算机主内存（如系统 **RAM**）
- 显卡内存
**宏观：**![[Pasted image 20260130193714.png]]
##### 行列复用
- 作用：减少地址线
- 芯片的外部只提供 **log₂(max(行数, 列数))** 根地址线。
- 地址分两次送入：
    1. 先送行地址（RAS）
    2. 再送列地址（CAS）
#### 对比表
| 特性   | DRAM         | SRAM       |
| ---- | ------------ | ---------- |
| 存储单元 | 1 晶体管 +1 电容器 | 4-6 晶体管    |
| 速度   | 较慢           | **更快**     |
| 功耗   | 刷新导致较高功耗     | 静态低，总体较高   |
| 成本   | 低            | 高          |
| 容量   | 大            | 小          |
| 刷新需求 | 需要 **定期刷新**  | 无需刷新       |
| 应用   | 主内存、显存       | CPU 缓存、寄存器 |
### Flash 存储
- **非易失性** ：利用半导体技术来存储数据，永久存储
作用场景：
- 长期数据存储
- USB 闪存驱动器
- 固态硬盘（SSD）
- 移动设备的内部存储
特性：
- 读取速度比 **RAM** 慢，但优于HDD
- 可读写
分类：**NAND Flash** 和 **NOR Flash** 
### ROM
- **非易失性**
- 早期不可写仅可读，后期可读写
**作用：**
- 常用于存储固件
- 软件程序直接嵌入在硬件设备中，以控制设备的基本操作，如BOIS
**分类：**
- EPROM （可写）与 CDROM （仅读），其中CDROM是光盘，采用串行存取方式，是唯独一个不采用随机存取方式的设备
#### 与RAM对比
|特征|RAM（随机存取存储器）|ROM（只读存储器）|
|---|---|---|
|易失性|是（断电时数据丧失）|否（数据在断电时不丧失）|
|读写能力|可以被随机读取和写入|通常只读，只读数据不能随机写入|
|用途|用于存储正在运行的程序、操作系统和数据|用于存储固定程序代码、固件、启动程序等|
|存储容量|通常较大，以支持多个程序同时运行|通常较小，主要用于存储核心系统和固件|
|数据持续性|不持久，数据在断电时丧失|持久，数据在断电时不丧失|
|修改权限|可以随机修改和写入数据|通常只读，只读数据无法被修改|
> [!warning] 做题须知
> 区分两种 MB：
> 
> **计算机存储容量的 MB（二进制）**：
   1 MB = $2^{20}$ Byte = 1,048,576 Byte
   常用于 **内存、ROM、显存、硬盘理论容量**（尤其在计组和 MAR 题里）
   >
   **数据传输速率的 MB（十进制）**：
   1 MB = $10^{6}$ Byte = 1,000,000 Byte
   常用于 **带宽 / 硬盘标称速度 / 网络速度**
   Mbps、MB/s 里往往默认十进制（工程计量单位）
### 错题
> 某计算机存储器按字节编址，主存地址空间大小为 64MB，现用 4M×8 位的 RAM 芯片组成 32MB 的主存储器，则存储器地址寄存器 MAR 的位数至少是（ ）
> 	A.22 位       B.23 位       C.25 位       D.26 位
- 实际的主存容量 32MB 不能代表 MAR 的位数，考虑到存储器扩展的需要，MAR 应保证访问到整个主存地址空间，因此MAR的表示内存实际为64MB
- 根据计算机存储换算方式 1MB = $2^{20}$ Byte ，MAR表示的实际内存为64*$2^{20}$ Byte
- 寄存器位数表示方式：$x$ Byte = $log_{2}x$ 位
- 由上述可得最终位数至少为26位
