
### 全局安装指定版本django

```
pip install django == 2.2.2
```

### 虚拟环境配置Django

1. 创建虚拟环境

```
python -m venv venv_name
```

2. 使用虚拟环境

```
cd venv_name\scripts
activate
```

3. 在虚拟环境中安装django

```
pip install django
```

### 创建启动项目.bat文件

1. 使用管理员模式打开记事本，输入类似以下代码

```
@echo off
D:
call D:\django_work\myDjango\venv\Scripts\activate.bat
cmd /k
```

- `echo off`：关闭命令行窗口中命令本身的显示（只显示输出，不显示命令行内容）。
- `@`：表示不显示这一行本身（就是不显示 `echo off` 这一行）。
- `call`：让批处理脚本去**调用另一个批处理文件**，执行完后还能继续回来运行原来的脚本。否则直接 `activate.bat` 会导致当前脚本“跳过去不回来”，后面的命令不执行。
- `**cmd /k**`：

- `/k`：执行完命令 **不关闭窗口**（keep）
- `/c`：执行完命令 **立刻关闭窗口**（close）

2. 将.txt文件后缀名改为.bat
3. 在打开后的命令行中输入启动项目

```
python manage.py runserver
```

4. 项目运行在8000端口，可在 http://127.0.0.1:8000 访问
