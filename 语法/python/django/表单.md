# 什么是“表单”？

在网页上，当你看到像这样👇的东西：

```
名字：[__________]
邮箱：[__________]
留言：[__________]

[提交]
```

这就叫 **表单（form）**，用户在这里输入内容，然后点击“提交”，数据就会传到后端。

---

## Django 表单是干嘛的？

Django 是后端框架，它也能自动帮你“画出这些表单”，并且：

- 帮你 **检查用户有没有乱输**（比如邮箱格式对不对）
- 帮你 **收集用户输入的内容**
- 帮你 **保存到数据库**（比如留言、注册用户、发帖）

---

## 举个例子：我要做一个“留言板”

我想做一个留言表单，用户填完点“提交”就能看到留言。

### ① 第一步：写一个表单类（`forms.py`）

```
from django import forms

class MyForm(forms.Form):
    name = forms.CharField(label="名字")
    message = forms.CharField(label="留言", widget=forms.Textarea)
```

👆 这段代码就相当于告诉 Django：“我要一个表单，有两个输入框，一个叫名字，一个叫留言。”

---

### ② 第二步：写一个视图函数（`views.py`）

```
from django.shortcuts import render
from .forms import MyForm

def my_view(request):
    if request.method == "POST":
        form = MyForm(request.POST)
        if form.is_valid():
            name = form.cleaned_data["name"]
            message = form.cleaned_data["message"]
            print("用户提交了：", name, message)
            # 你也可以保存数据、发邮件、跳转等等
    else:
        form = MyForm()

    return render(request, "my_template.html", {"form": form})
```

---

### ③ 第三步：写 HTML 页面（`my_template.html`）

```
<form method="post">
    {% csrf_token %}
    {{ form.as_p }}
    <button type="submit">提交</button>
</form>
```

### 运行效果是：

- 用户看到两个输入框
- 用户输入完点提交
- 你就能在后台拿到数据（比如打印出来、存数据库）
