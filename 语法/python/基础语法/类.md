### self：
- 一个对象的方法被调用的时候，对象会将自身的引用作为第一个参数（即方法的self参数）
- 也就是说在定义类中的方法时必须写self参数
- ‘self’也可以改为‘my’等
```py
class P:
    def named(self,name):
        self.name = name
xian = P()
xian.named('先')
#'先'传递给name
```
```py
class Apa:
    def aa(self):
        print()
Apa.aa()
#类名调用会报错，self无参数传递
```

### 类方法，静态方法和实例方法：
- `cls`：相当于self，传参
在 Python 中，cls() 是用来调用当前类的构造函数（__init__方法）并创建该类的一个新实例。
#### 详细说明
1. cls 是类方法中的参数：
- 当方法被 @classmethod 装饰时，方法的第一个参数是类本身，而不是实例。
- 通常将这个参数命名为 cls，类似于实例方法中的 self。
2. cls() 的含义：
- 调用 cls() 实际上是在调用该类的构造函数 **init**，并创建一个新的实例。
- 如果 **init** 定义了参数，那么 cls() 需要传递相应的参数。
---

示例
`class Student:  
`def init(self, name="Default"):  
`self.name = name`
```py
@classmethod
def create_default(cls):
    # cls() 调用构造函数 __init__，创建一个实例
    return cls()

@classmethod
def create_with_name(cls, name):
    # cls(name) 调用 __init__，传递参数
    return cls(name)
```
## 创建默认实例
default_student = Student.create_default()  
print(default_student.name) # 输出：Default
## 创建带有名字的实例
named_student = Student.create_with_name("Alice")  
print(named_student.name) # 输出：Alice

## 封装，继承与多态：
**封装**：私有化变量或方法时只需在名字前加两个下划线“__”。私有变量无法被直接访问，需要通过接口。
```py
class person:
    def __init__(self,age):
            self.__age = age
    def __sayHello(self):
            ……
    def greating(self):
            self.__sayHello()
aleee = person(12)
aleee.greating()
```
python的私有化变量是伪私有，可以使用“`_类__属性`”访问
- **继承**：子类可以继承父亲的属性和行为，可以改变从父类继承来的行为
_单继承_：只有一个父类
```py
class Cat(Animal):
        pass
```
_多继承_：继承了多个父类
```py
class Son(Dad,Mom):
        pass
```
如果Dad与Mom中有同名方法，会先继承Dad类
- **方法重写**：可以在子类中重写父类方法，子类中的方法会覆盖父类。在子类中调用父类方法可以使用`super()`函数，或使用“父类名.方法名()”来实现。
```python
class Student:
        def __init__(self,name):
                self.name = name
……
class Pupil(Student):
        def __init__(self,name,age):
                super().__init__(name)
                self.age() = age
……
```
- **多态**：向不同的对象发送同一条消息，不同的对象在接收时会产生不同的行为/方法。
