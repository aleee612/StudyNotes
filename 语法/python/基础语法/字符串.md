## 字符串格式化：
### **_%_**：
```python
#'%[+-][0][m][.n]格式字符' %变量

print('a = %0.5f'%a)
```
- +- ：+为右对齐输出，-为左对齐输出，默认右对齐
- 0：0为补0，缺省补空格
- m：输出的最小位数
- .n：小数位数
### _format()：_
```python
#模板字符串.format(逗号分隔的参数)

s = 'hello,{1},you have{0}.'.format(500,aleee)

s = 'hello,{},you have{}.'.format(aleee,500)
```
- `format()`中用逗号分隔的参数按序号替换到模板字符串中的槽中（从0开始）
- {参数序号:格式控制标记}
格式控制标记：
```python
num = 42
print(f"{num:05}")  # 输出: 00042 (左填充，宽度为5，用0填充)
print(f"{'Hi':<5}") # 输出: Hi   (右填充，宽度为5，用空格填充)
```

|   |   |
|---|---|
|填充|用于填充的单个字符|
|,|数字的千位分隔符|
|对齐|<左对齐 >右对齐 ^居中|
|宽度|输出宽度|
|.精度|浮点数小数部分精度或字符串最大输出长度|
|格式字符|整数类型d、o、x、X、b，浮点数类型e、E、f，百分比%|

## 字符串的常用操作：
### 查找：
```python
str.find(sub[,start[,end]])
```
- `sub`：要查找的字符串
- `start`：开始索引，缺省为0
- `end`：结束索引，缺省为字符串长度
查找到返回索引，未找到返回-1
### 分割字符串：
```python
str.split([delimiter,num])

str.split(' ',3)
```
- `delimiter`：删去的字符
- `num`：分割的次数
### 连接：
```python
str.join(sequence)

'-'.join("Python")
```
- `str`：连接符，可以为空
- `sequence`：要连接的序列
### 替换：
```python
str.replace(old,new[,max])
```
- `old`：被替换的
- `new`：替换的
- `max`：最大替换次数，缺省为全部
### 移除首尾字符：
```python
str.strip([chars])

str.strip('10') #移除首尾所有的1和0
```
- chars：要移除的指定字符，缺省时为空格
### 转换字符串的大小写：
```python
str.lower()
str.upper()
```
### 将单词首字母转化为大写：
```python
str.title()
```
### 检测字符串：
检测是否由数字和字母组成，是则返回True，否则返回False
```python
str.isalnum()
```
