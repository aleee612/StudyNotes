
## 打开与关闭
### 打开
```python
# open(filename[, mode, encoding = ''])

file = open('filename')
```
- `mode`：读写模式，缺省为只读，数据类型为字符串
- `encoding = ‘UTF-8’`时可以使用中文
文件打开模式

|   |   |   |
|---|---|---|
|r|只读|抛出异常|
|w|只写|新建文件|
|a|末尾追加|新建文件|
|rb|二进制，只读|抛出异常|
|wb|二进制，只写|新建文件|
|ab|二进制，末尾追加|新建文件|
|r+|读写|抛出异常|
|w+|读写|新建文件|
|a+|读，末尾追加|新建文件|
|rb+|二进制，读写|抛出异常|
|wb+|二进制。读写|新建文件|
|ab+|二进制，读，末尾追加|新建文件|

### 关闭
```python
file.close()
```
### with语句
```python
with context_expression [as target(s)]:
        with_body

with open(filename[,mode]) as file:
        with_body
```
## 读写文件
### 写入（两种写入方式都不会自动换行）
`write()`_：写入的为字符串_
```python
file.write(str)
```
- 当使用‘b’模式打开文件，需使用encode()方法将字符串转变为字节流形式
```python
file = open('a.txt',wb)
file.write('aleee'.encode())
```
`writelines()`：写入的为字符串序列，如列表
```python
file.writelines(sequence)
```
### 读取文件
`read()`_：读取整个文件_
```python
file.read([size])
```
- size：从文件中读取的字节数，缺省或为负则返回整个文件，换行符也为一个字节
`readline()`*：*从文件中读取一行，包括换行符
```python
file.readline([size])
```
`readlines()`：读取所有行直到结束符EOF并返回列表，列表元素为每一行的数据
```python
file.readlines()
```
### 定位文件
`tell()`：获取当前读写位置（指针的位置，即从头文件开始的字节数）
```python
file.tell()
```
`seek()`：移动文件指针到指定位置
```python
file.seek(offset[,from])
```
- `offset`：偏移量
- `from`：起始点，有三种取值：

|   |   |
|---|---|
|0|文件开始位置|
|1|当前位置|
|2|文件末尾位置|

- 当offset和from都为0时，文件必须从二进制模式打开，否则抛出异常
