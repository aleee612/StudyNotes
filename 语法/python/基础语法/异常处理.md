## 常见内置异常exception

|                     |                   |
| ------------------- | ----------------- |
| `TypeError`         | 不同类型的数据进行运算操作     |
| `ZeroDivisionError` | 除数为0              |
| `NameError`         | 尝试访问一个未声明的变量      |
| `IndexError`        | 索引超出序列的范围         |
| `IndentationError`  | 缩进错误              |
| `KeyError`          | 使用字典中不存在的键        |
| `ValueError`        | 传给函数的参数类型不正确      |
| `FileNotFoundError` | 试图以只读模式打开一个不存在的文件 |
| `AttributeError`    | 尝试访问未知的对象属性       |
| `SyntaxError`       | 语法错误              |

## （多重）try - except语句
```python
try:
        # 代码块
except exception1[as e]:
        # 输出异常
[
except (exception2,exception3):
        # 捕获多种异常
except:
        # 不指定异常类型，输出所有异常
]
```
- exception：异常类名
- as e：将异常类的实例对象名命名为e
- 多重except语句会依次检查每一个except语句，并执行第一个异常类型匹配的代码块，其他的except语句将不再捕获
### try - except - else语句
```python
try:
        # 代码块
except exception1[as e]:
        # 输出异常
else:
        # 若无异常则执行该代码块
```
### try - except - finally语句
```python
try:
        # 代码块
except exception1[as e]:
        # 输出异常
finally:
        # 无论是否异常都会执行代码块
```
## 主动抛出bug
### raise语句
```python
raise [exception[(message)]]
try:
        if n > 0:
                raise Exeption('不能大于0')
except Exception as e:
        print('异常信息：',e)
```
- message：输出的异常语句
- exception，message都缺省时将重新引发它（传递异常）
### assert语句
```python
assert expression [,message]

# 等同于
if not exception:
        raise AssertionError(message)
        
try:
        assert i < 2,'i不能小于2'
except Exception as e:
        print('异常信息：',e)
```
