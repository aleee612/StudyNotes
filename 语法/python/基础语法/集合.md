## 集合的基本操作：
### 添加和删除：
```python
# 添加
s.add(x)
# 合并集合T的元素到s并自动去除重复
s.update(T)
# 删除末尾，集合为空则抛出异常
s.pop()
# x在s内则删除，否则抛出异常
s.remove(x)
# x在s内则删除，否则也无报错
s.discard(x)
#清空集合
s.clear()
```
### 集合的运算：
```python
s & t   # 交集
s | t   # 并集
s - t   # 差集
s ^ t   # 补集
s <= t  # s为t的子集或s=t，返回True
s >= t  # t为s的子集或s=t，返回True
```
