- 列表要处理一组数据，因此必须通过显式的数据赋值才能生成，简单的将一个列表赋值给另一个列表不会产生新的列表对象，只会引用。
## 创建与访问：
```python
animal = ['elephant','……',……]
```
### 其他序列 _转换_ 为列表（如字符串，range对象，元组等）：
```python
str1 = list(str)
str2 = list('hello world')
str3 = list(range(1,10,2))
```
## 列表的遍历：
### _while:_
```python
while i < len(list):
        print(list[i])
        i += 1
```
### _for:_
```python
for item in list:
        print(item)
```
## 列表的基本操作：
### _添加：_
`append()`：末尾添加元素（对象）*
```python
list.append(obj)
```
`extend()`：在末尾添加另一个序列的多个值（序列）*
```python
list.extend(seq)
```
`insert()`：插入至指定位置（对象）*
```python
list.insert(index,obj)
```
### _删除_：
`del()`：索引删除，也可以使用分片*
```python
del list[index]

del list[a:b]
```
`pop()`：索引删除并返回元素的值，缺省时为-1*
```python
list.pop([index])
```
`remove()`：删除列表中的第一个适配项*
```python
list.remove(obj)
```
### _分片赋值：_
```python
list[a:b] = list2
#若a=b，则意味着在此处插入元素
```
### _列表排序：_
`reverse()`：将列表中的值反向存放（无参数无返回值）*
```python
list.reverse()
```
`sort()`：将列表中的元素排序（默认升序）*
```python
list.sort([key = None][,reverse = False])
```
- `key`：参考的方法，如len等
- `reverse`：是否为降序，True为降序
`sorted()`：返回新列表，不对原列表进行更改*
```python
sorted(iterable[,key = None][,reverse = False])
```
- `iterable`：可迭代对象
