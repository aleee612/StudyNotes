- 字典中的键是唯一的，若出现新的则覆盖前一个的值
## 字典的创建：
### 直接赋值：
```python
list = {key1:val1,key2:val2}
```
### 内置函数：
- `dict()`：
```python
list = dict(list1)
list = dict([(key1,val1),(key2,val2)])
list = dict(key1 = val1,key2 = val2)
list = dict(zip([key1,key2][val1,val2]))
```
- zip()：将可迭代的对象作为参数，将对象中的元素打包成一个个元组，然后返回这些元组组成的zip对象
- `fromkey()`：所有的键都对应一个值时
```python
dict.fromkey(seq[,value])
```
- `seq`：键的列表
- `value`：所有键对应的同一值，缺省为None
### 字典的访问：
根据 _键值_ 访问：
```python
dict[key]
```
- `get()`：若键存在则返回对应的值，不存在则返回默认值（default的值）
```python
dict.get(key[,default = None])
```
### 字典的基本操作：
_修改和添加：_
```python
dict['key'] = value
```
- 若key存在，则为改
- 若key不存在，则为增
_删除_：
`del`：删除单个
```python
del dict[key]
```
`clear()`：删除所有
```python
dict.clear()
```
`pop()`：删除指定并返回删除的值，default为默认值，若无指定的键则返回default
```python
dict.pop(key[,default])
```
`popitem()`：只能删除最后一个键值对
```python
dict.popitem()
```
_更新：_
```python
dict.update(dict1)

#e.g.
stu.update({'num':123,'name':'aleee'}) #一次性全部添加
```
_复制：_
```python
dict_ = dict
dict_ = dict.copy()
dict_ = copy.deepcopy(dict)
```
##### 字典的复制：
直接复制：复制对象的引用
- 浅复制：使用`copy()`方法复制父对象及其子对象的引用
- 深复制：使用`deepcopy()`方法完全复制父对象及其子对象（无引用），包含在copy模块中
_遍历：_
`item()`：输出键值对
```python
dict.item()
```
`key()`：输出键
```python
dict.keys()
```
`value()`：输出值
```python
dict.value()
```
