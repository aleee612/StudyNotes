### 索引：
- 从零开始，有负值，倒叙，末尾为-1
### 分片：
- 范围：[- len, len-1]
```python
slice[start:end:step]
```
- 相加：
```python
str3 = str1 + str2
```
- 相乘：
```python
str1 = str * 3
```
- 长度，最小值与最大值：
```python
str_len = len(str)
str_max = max(str)
str_min = min(str)
```
### 查找与计数：
`count：`出现次数
`index：`首次出现的位置（索引位置）
```python
index = str.index(obj)
num = str.count(obj)
```
