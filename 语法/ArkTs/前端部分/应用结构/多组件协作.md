## 基本用法

```
@Component
  struct ChildComponent {
    build() {
      Text("我是子组件")
        .fontSize(18)
    }
  }

@Entry
  @Component
  struct MyApp {
    build() {
      Column() {
        Text("我是主组件")
        ChildComponent()  // ✅ 使用子组件
      }
    }
  }
```

- 可以写多个 `@Component struct`，每一个都是可复用的小组件。
- 只有一个组件（ 通常是最顶层的页面 ）需要加 `@Entry`。
- 子组件**不能加** `**@Entry**`，否则系统不知道该从哪个开始跑。
