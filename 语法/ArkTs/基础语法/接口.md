- 用来**定义对象或类的结构规范**

- 开发者可以明确规定一个类必须实现的方法和属性

---

### 接口定义

```TS
interface PersonInterface {
  name: string;
  age: number;
  sayHello(): void;
}
```

- 接口无需（也没有）构造函数

- 只能包含**属性和方法的定义**，**不能写实现**

### 类实现接口

```TS
class Person implements PersonInterface {
  name: string;
  age: number;

  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }

  sayHello(): void {
    console.log(`Hi, I'm ${this.name}`);
  }
}
```

- 类可以 实现一个或多个接口，**必须实现接口中定义的所有成员**

### 接口也可以被对象使用

```TS
let student: PersonInterface = {
  name: "Tom",
  age: 18,
  sayHello() {
    console.log("Hello!");
  }
};
```

### 接口扩展（继承）

- 接口之间也可以继承，用 `extends`

```TS
interface Animal {
  name: string;
}

interface Dog extends Animal {
  breed: string;
}
```
