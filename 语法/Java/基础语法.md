# 基本数据类型
## 关键字&保留字
- Java的所有关键字都是小写
- 与c相同，不能作为变量/方法名使用
- Java的标识符可以使用带有`$`
---
## 数据类型
##### 1. 常量
- 命名方式：`final` = c/cpp中的`const`
##### 2. 变量
- Java的变量与c/cpp不同，它的长度是确定的，而c/cpp的整型会随着平台的不同而发生变化
- Java的布尔类型是`boolean`
- Java的字符型占2字节，而c/cpp的只占1字节
- Java的`long`相当于c/cpp的`long long`，固定为64位
##### 3.[[大数]]
- 在基本的整数和浮点数都不能满足情况时使用，比如数据过大时

---
## 引用数据类型
- Java中对象（类、[[语法/Java/数组]]、接口等）可能非常大，直接把它们当值传来传去会很慢；
- 对象存放在堆内存里，变量存放的是引用（类似指针）。
- Java 保留了 8 个基本数据类型（`byte, short, int, long, float, double, char, boolean`），它们是 **值类型**，直接存储在栈或[[寄存器]]里，效率高。

# 基本语句

## switch语句扩展

- 箭头case，可以跟表达式后面的值
- `yield someValue`中somevalue表示yield所在case分支产生的switch表达式的值

```java
int j = switch(day){
    case Monday -> 1;  //j = 1
    case Tuesday -> 2; //j = 2
    default->{
        int result = 3;
        yield result;  //j = 3
}}
```

- 循环语句可以使用`labor`指定跳转到的语句
