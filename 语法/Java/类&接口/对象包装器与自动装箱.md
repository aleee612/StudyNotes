### 对象包装器
- 所有基本类型都有一个与之对应的类，这些类称为包装器
- Integer，Long，Float，Double，Short，Byte，Character和Boolean
- 前六位派生于Number
- 对象包装器中的对象不可变
- 当使用对象包装器的算数表达式时也需要使用自动拆箱与装箱
- 由于对象包装器的本质是类，因此无法使用`==`做相等对比，最好是使用`equals`
- 如果是Integer升Double也需要先拆箱再转型再装箱
- 装箱和拆箱是编译器的操作而非虚拟机
### 自动装箱
```java
list.add(3);
list.add(Integer.valueof(3));
```
- 上述两式效果相等，而1式子会自动转换成2，这种操作称为自动装箱
### 自动拆箱
```java
int n = list.get(i);
=> int list.get(i).intValue();
```
