- 将类组织在一个集合中
### 包名
- 因特网域名的逆序形式
- 如：com.horstmann
- 可以在其后追加工程名和类名，如：com.horstmann.corejava.Emp就是一个完全限定名
- 从编译器的角度来看，嵌套的包之间没有任何关系。e.g：java.util包与java.util.jar包毫无关系，每一个包都是独立的类集合
### 类的导入
1. 使用完全限定名
```java
java.time.LocalDate today = java.time.LocalDate.now();
```
2. 使用import导入包中的类
```java
import java.time.*;//所有类
import java.time.LoacalDate;//单个类
```
- `*`只能用于导入一个包，而不能使用`import java.*`或`import java.*.*`这种方法
- 当导入的包中有相同类名时需要在类名前加上完整的包名
- 可以认为java中的package和import语句类似于c++中的namespace和using指令
### 静态导入
- 导入静态方法和静态字段而不只是类
```java
import static java.lang.System.*;
import static java.lang.System.out;//导入特定方法或字段
...
out.println("Good");
exit(0);
```
- 就不必添加前缀了
### 在包中增加类
1. 在开头写包名
```java
package com.hostmann.corejava;
public class Emp
{
	...
}
```
- 如果没有在源文件中写package语句则该类属于无名包。无名包顾名思义，没有包名。
2. 将源文件放到与完整包名匹配的子目录中
