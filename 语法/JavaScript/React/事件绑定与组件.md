# 事件绑定

- 语法：`on + 事件名称 = { 事件处理程序 }`
- 事件绑定了就会调用该函数

```
function App() {
  const handleClick = () => {
    console.log('click');
  }
  return(
    <div className="App">
      <button onClick={ handleClick }>click me ^Q^</button>
    </div>

  )
}
```

```
  //传递自定义参数
  //事件参数e
  const handleClick = (name,e) => {
    console.log('click',name,e);
  }
  return(
    <div className="App">
      <button onClick={(e) => handleClick('aleee',e)}>click me ^Q^</button>

    </div>

  )
```

# 组件基础使用

组件：**首字母大写**的函数，存放了组件的逻辑和视图ui，渲染组件只需要把组件当成标签书写即可

```
function Button(){ //定义
    //业务逻辑组件逻辑
  return <button>click me</button>

}

function App() {      //使用
  return (
    <div className="App">
      {/* 自闭合 */}
      <Button />
      {/* 成对标签 */}
      <Button></Button>
    </div>
  );
}
```

# 渲染页面

- 使用要渲染的`function`，并且在文件末尾写上`export default Name`（注意Name首字母一定要大写，否则 `React` 会把它当成普通 HTML 标签看待，不会渲染它。）
