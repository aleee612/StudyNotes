# 抽象类（Abstract Class）

- 包含**至少一个纯虚函数**（pure virtual function）的类

- 不能实例化对象，只能作为**基类被继承**

- 纯虚函数以 `= 0` 结尾

```
class Animal {
public:
    virtual void speak() = 0; // 纯虚函数
};
```

---

## 继承抽象类

- 子类必须实现所有的纯虚函数才能实例化对象

```
class Dog : public Animal {
public:
    void speak() override {
        std::cout << "Woof!" << std::endl;
    }
};
```

---

## 虚函数（Virtual Function）

- 用 `virtual` 关键字修饰

- 支持**多态**，即通过基类指针调用子类实现的方法

```
class Animal {
public:
    virtual void speak() {
        std::cout << "Animal speaks" << std::endl;
    }
};

class Cat : public Animal {
public:
    void speak() override {
        std::cout << "Meow" << std::endl;
    }
};

void makeSpeak(Animal* animal) {
    animal->speak(); // 运行时根据实际类型调用对应方法
}
```

---

## 示例：抽象类 + 多态

```
#include <iostream>
using namespace std;

class Animal {
public:
    virtual void speak() = 0; // 纯虚函数，抽象类
};

class Cat : public Animal {
public:
    void speak() override {
        cout << "Meow" << endl;
    }
};

class Dog : public Animal {
public:
    void speak() override {
        cout << "Woof" << endl;
    }
};

int main() {
    Animal* a1 = new Cat();
    Animal* a2 = new Dog();

    a1->speak(); // Meow
    a2->speak(); // Woof

    delete a1;
    delete a2;
    return 0;
}
```
