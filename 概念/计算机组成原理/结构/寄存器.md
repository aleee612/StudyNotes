- 由硬件直接构造，数量有限
- 位于[[cpu]]中
### 种类
#### MIPS-32架构
| 通用寄存器     | 别名        | 用途                          |
| --------- | --------- | --------------------------- |
| `$0`      | `zero`    | 恒为 0，不能写入                   |
| `$1`      | `at`      | 汇编器临时用（Assembler Temporary） |
| `$2-$3`   | `v0-v1`   | 函数返回值（Value）                |
| `$4-$7`   | `a0-a3`   | 函数参数（Argument）              |
| `$8-$15`  | `t0-t7`   | 临时寄存器，函数调用可能被覆盖             |
| `$16-$23` | `s0-s7`   | 保存寄存器，函数调用要保存原值             |
| `$24-$25` | `t8-t9`   | 临时寄存器                       |
| `$26-$27` | `k0-k1`   | 内核使用                        |
| `$28`     | `gp`      | 全局指针（Global Pointer）        |
| `$29`     | `sp`      | 堆栈指针（Stack Pointer）         |
| `$30`     | `fp`/`s8` | 帧指针或保存寄存器                   |
| `$31`     | `ra`      | 返回地址（Return Address）        |

| 特殊寄存器               | 用途               |
| ------------------- | ---------------- |
| `HI` / `LO`         | 存放乘法、除法结果        |
| `PC`                | 程序计数器，存放下一条指令地址  |
| `Status`, `Cause` 等 | 控制和状态寄存器，用于异常和中断 |
