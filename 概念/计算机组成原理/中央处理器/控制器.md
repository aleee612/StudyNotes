### 控制器
#### 硬布线控制器
##### 原理
- 操作码译码
- 时序系统产生控制信号
- 组合逻辑电路实现控制
##### 特点
| 优点       | 缺点      |
| -------- | ------- |
| 速度快      | 设计复杂    |
| 执行效率高    | 扩展和修改困难 |
| 适用于高速处理器 | 规整性较差   |

#### 微程序控制器

##### 概念
| 名称        | 含义               |
| --------- | ---------------- |
| 微命令       | 控制部件向执行部件发出的控制信号 |
| 微操作       | 控制信号使部件完成的操作     |
| 微指令       | 若干微命令的集合         |
| 微程序       | 实现一条机器指令功能的微指令序列 |
| 控制存储器（CM） | 存放微程序            |
##### 组成和工作过程
###### 组成
- 控制存储器 CM
- 微地址寄存器 μAR
- 微指令寄存器 μIR
- 地址转移逻辑
###### 工作过程
1. μAR给出地址；
2. 从控制存储器取出微指令；
3. 微指令送入μIR；
4. 发出控制信号完成微操作；
5. 地址转移逻辑形成下一微地址；
6. 执行下一条微指令。

---
##### 微指令编码方式
| 编码方式   | 特点            | 优点          | 缺点      |
| ------ | ------------- | ----------- | ------- |
| 直接编码   | 一位对应一个微命令     | 执行速度快，并行能力强 | 微指令字长较长 |
| 字段直接编码 | 一个字段对应一组互斥微命令 | 缩短字长        | 需译码     |
| 字段间接编码 | 二次译码          | 字长最短        | 执行速度最慢  |
###### 比较
| 项目    | 直接编码 | 字段直接编码 | 字段间接编码 |
| ----- | ---- | ------ | ------ |
| 微指令字长 | 长    | 较短     | 最短     |
| 控存容量  | 大    | 较小     | 最小     |
| 执行速度  | 快    | 较快     | 慢      |
| 并行能力  | 强    | 较强     | 较弱     |

---

##### 后继微地址形成方式
| 形成方式   | 原理           |
| ------ | ------------ |
| 顺序方式   | μAR+1        |
| 断定方式   | 由微指令下地址字段给出  |
| 操作码方式  | 由机器指令操作码形成   |
| 条件转移方式 | 根据状态条件决定转移地址 |
| 混合方式   | 多种方式结合       |

---
##### 微指令格式
###### 水平型与垂直型比较
| 项目    | 水平型       | 垂直型      |
| ----- | --------- | -------- |
| 控制方式  | 一位对应一个微命令 | 类似机器指令   |
| 并行能力  | 强         | 弱        |
| 微指令字长 | 长         | 短        |
| 控存容量  | 大         | 小        |
| 执行速度  | 快         | 慢        |
| 编码译码  | 简单        | 复杂       |
| 适用场景  | 高速系统      | 控存容量受限系统 |
###### 混合型微指令
- 兼有水平型和垂直型特点；
- 在速度和字长之间折中。

---
#### 两种控制器比较
| 项目       | 硬布线控制器 | 微程序控制器 |
| -------- | ------ | ------ |
| 控制信号产生方式 | 组合逻辑电路 | 微指令    |
| 执行速度     | 快      | 较慢     |
| 设计复杂度    | 高      | 低      |
| 规整性      | 差      | 好      |
| 扩展和修改    | 困难     | 容易     |
| 适用机器     | RISC   | CISC   |
