### 不能
| 东西                | 在哪一层     | 谁能看见               |
| ----------------- | -------- | ------------------ |
| **ULT（用户级线程）**    | 用户态      | 线程库（如 pthread、JVM） |
| **KLT（内核级线程）**    | 内核态      | 操作系统内核             |
| **CPU 用户态 / 内核态** | CPU 执行状态 | 硬件                 |
- ULT / KLT 是“软件抽象”
- 用户态 / 内核态 是“CPU 执行模式”
**这是两条完全不同的维度**
#### CPU 有用户态 / 内核态
#### KLT、ULT 本身没有“态”
永远只是一个**数据结构**：
- TCB
- 栈
- 寄存器快照（保存在用户空间）
