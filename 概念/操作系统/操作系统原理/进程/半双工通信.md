- 半双工通信（Half-Duplex Communication）是一种通信方式
---
### 定义
- **信息在两个方向都可以传输**，但是 **同一时刻只能有一个方向在传输**。
- 换句话说，发送和接收是 **轮流进行** 的，而不能同时进行。
---
### 特点

| 特性   | 说明                         |
| ---- | -------------------------- |
| 传输方向 | 双向，但不能同时传输                 |
| 典型设备 | 对讲机、集线器（Hub）等              |
| 优点   | 节省传输线路成本                   |
| 缺点   | 通信效率低于全双工，必须等待对方完成发送才能轮到自己 |

---
### 对比
- **单工（Simplex）**：信息只能单向传输（例如电视广播）
- **半双工（Half-Duplex）**：信息可以双向，但不能同时（例如对讲机）
- **全双工（Full-Duplex）**：信息可以双向同时传输（例如电话、网线通信）
---
### 举例
- **对讲机**：你按下按钮说话，对方才能听到；松开后，对方才可以说话
- **以太网 Hub**：半双工模式下，数据包发送或接收不能同时发生
