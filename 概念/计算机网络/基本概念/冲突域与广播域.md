### 冲突域与广播域
#### 1. 冲突域
> 多个设备竞争同一条信道、可能发生数据碰撞的范围。
###### 哪些设备能分割冲突域
| 设备            | 是否分割冲突域 |
| ------------- | ------- |
| 中继器（Repeater） | ×       |
| 集线器（Hub）      | ×       |
| 网桥（Bridge）    | √       |
| 二层交换机（Switch） | √       |
| 路由器（Router）   | √       |
| 三层交换机         | √       |
###### 规律
- 二层及以上设备都能分割冲突域
- 交换机每个端口对应一个冲突域

---
#### 2. 广播域
> 广播帧能够传播到的范围。
###### 哪些设备能分割广播域
| 设备            | 是否分割广播域 |
| ------------- | ------- |
| 中继器（Repeater） | ×       |
| 集线器（Hub）      | ×       |
| 网桥（Bridge）    | ×       |
| 二层交换机（Switch） | ×       |
| 路由器（Router）   | √       |
| 三层交换机         | √       |
| VLAN          | √       |
###### 规律
- 三层设备才能分割广播域
- 一个 VLAN 对应一个广播域

---
### 各设备对域的影响
| 设备           | 分割冲突域            | 分割广播域 |
| ------------ | ---------------- | ----- |
| 中继器          | ×                | ×     |
| 集线器 Hub      | ×                | ×     |
| 网桥 Bridge    | √                | ×     |
| 二层交换机 Switch | √                | ×     |
| 路由器 Router   | √                | √     |
| 三层交换机        | √                | √     |
| VLAN         | —（VLAN 本身不是网络设备） | √     |
>[!note]
一层不隔离，
二层隔冲突，
三层隔广播。
Hub 不隔离任何域；
Switch 隔离冲突域；
Router 隔离广播域；
一个 VLAN 就是一个广播域。
