### 一般顺序
```bash
HELO/EHLO → MAIL FROM → RCPT TO → DATA → 邮件正文 → . → QUIT
```
---
### 会话管理命令

| 命令       | 作用                          | 示例               | 服务器返回                                           |
| -------- | --------------------------- | ---------------- | ----------------------------------------------- |
| **HELO** | 客户端向服务器打招呼，报自己身份            | `HELO crepes.fr` | `250 Hello crepes.fr`                           |
| **EHLO** | 扩展 HELO，支持 SMTP 扩展功能（认证、加密） | `EHLO crepes.fr` | `250-server.example.com Hello crepes.fr` + 支持列表 |
| **QUIT** | 结束会话，断开连接                   | `QUIT`           | `221 server.example.com closing connection`     |

---

### 邮件发送相关

|命令|作用|示例|服务器返回|
|---|---|---|---|
|**MAIL FROM:**|指定发件人|`MAIL FROM:<alice@crepes.fr>`|`250 Sender ok`|
|**RCPT TO:**|指定收件人|`RCPT TO:<bob@hamburger.edu>`|`250 Recipient ok`|
|**DATA**|开始输入邮件正文，输入 `.` 结束|`DATA` → 输入正文 → `.`|`354 Start mail input` → `250 Message accepted`|
|**RSET**|重置当前邮件事务（取消正在发送的邮件）|`RSET`|`250 OK`|
|**VRFY**|验证邮箱地址是否存在（不一定启用）|`VRFY bob@hamburger.edu`|`250 bob@hamburger.edu` 或 `550 User unknown`|
|**EXPN**|展开邮件列表（不一定启用）|`EXPN staff-list@domain.com`|`250 user1@domain.com …`|

---

### 服务器特性查询

|命令|作用|示例|服务器返回|
|---|---|---|---|
|**NOOP**|测试服务器是否在线，不做其他操作|`NOOP`|`250 OK`|
|**HELP**|查看服务器支持的命令|`HELP`|列出命令列表|
|**STARTTLS**|请求开启加密连接（TLS/SSL）|`STARTTLS`|`220 Ready to start TLS`|
