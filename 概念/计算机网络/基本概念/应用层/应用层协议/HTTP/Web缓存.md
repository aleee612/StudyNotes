- 又名代理服务器
- 用于表示初始服务器来满足HTTP请求的网络实体
### 步骤
1. 浏览器创建到Web缓存的TCP连接，并向Web缓存区中的对象发送HTTP请求
2. Web缓存器检查本地是否有对象副本，若有则发送响应报文返回对象
3. 若无则打开一个与该对象的初始服务器的TCP连接，发送请求并得到响应
4. 缓存器接收对象时创建副本，并向客户发送副本
- 由此可见，Web缓存器既是服务器也是客户
### 来源
- 由[[ISP]]购买和安装
### 作用
- 大大减少对客户请求的响应时间
- 大大减少一个机构的接入链路到因特网的通信量（流量），就不必急于增加带宽（省钱）
### 使用技术
- [[内容分发网CDN|内容分发网络]]（CDN）
- 证实所存储对象是最新的：[[条件GET方法]]
