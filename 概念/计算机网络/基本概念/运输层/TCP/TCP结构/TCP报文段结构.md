- 当TCP发送一个大文件时，通常是将文件分为长度是[[MSS最大报文长度|MSS]]的若干块
- 首部：
	- [[序号和确认号|32bit序号+32bit确认号]] => 实现[[可靠数据传输原理|可靠数据传输服务]]
	- 16bit接收窗口 => 用于流量控制（指示接收方愿意接收的字节数）
	- 4bit首部长度 => 首部长度 = 字段值 × 4 字节
	- 选项字段 => 协商[[MSS最大报文长度|MSS]] 、高速网络环境用作窗口调节因子
	- [[TCP 6bit 标志位|6bit标志字段]] 
	- 最小首部 = 20B；最大首部：60B
