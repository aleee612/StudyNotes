### 端口号结构 & 范围
- 是一个16比特的数
- 总范围为 0 ~ 65535
### 常见应用层协议端口号

| 协议                 | 端口号             | 协议类型    | 说明                  |
| ------------------ | --------------- | ------- | ------------------- |
| **HTTP**           | 80              | TCP     | 浏览网页的标准端口           |
| **HTTPS**          | 443             | TCP     | 加密的 HTTP（SSL/TLS）   |
| **FTP**            | 21              | TCP     | 文件传输控制（登录与命令）       |
| **FTP-Data**       | 20              | TCP     | 文件数据传输              |
| **SSH**            | 22              | TCP     | 安全远程登录（替代 telnet）   |
| **Telnet**         | 23              | TCP     | 早期远程登录（不安全）         |
| **SMTP**           | 25              | TCP     | 发送电子邮件（邮件传出）        |
| **POP3**           | 110             | TCP     | 接收邮件（早期协议）          |
| **IMAP**           | 143             | TCP     | 邮件接收（更先进）           |
| **DNS**            | 53              | UDP/TCP | 域名解析（UDP为主，TCP用于大包） |
| **DHCP**           | 67（服务器）/68（客户端） | UDP     | 动态分配 IP 地址          |
| **SNMP**           | 161             | UDP     | 网络设备监控协议            |
| **TFTP**           | 69              | UDP     | 简化文件传输协议            |
| **NTP**            | 123             | UDP     | 网络时间同步              |
| **HTTP Proxy（代理）** | 8080            | TCP     | 常见的代理服务器端口          |
| **MySQL**          | 3306            | TCP     | 数据库服务               |
| **PostgreSQL**     | 5432            | TCP     | 数据库服务               |
| **RDP**            | 3389            | TCP     | Windows 远程桌面连接      |
| **Redis**          | 6379            | TCP     | 内存数据库               |
| **MongoDB**        | 27017           | TCP     | 文档型数据库              |

---
### 端口号分类（按范围）

| 范围                              | 名称          | 范围号                             |
| ------------------------------- | ----------- | ------------------------------- |
| **知名（周知）端口（Well-known Ports）**  | 0–1023      | 固定分配给常见[[应用层协议]]（HTTP、FTP、DNS等） |
| **注册端口（Registered Ports）**      | 1024–49151  | 软件厂商申请使用（如 MySQL 3306）          |
| **动态端口（Dynamic/Private Ports）** | 49152–65535 | 系统临时分配，用于客户端通信                  |
