- 作用：改变[[概念/计算机网络/基本概念/物理层/基本概念#码元|码元]]携带数据大小
- 方式：设置**模拟信号**中信号的频率、相位、振幅 等
#### 调制方法
> **数字信号** -> **模拟信号** 
![[Pasted image 20260206224619.png|438]]

| 调制类别  | 调制名称       | 英文缩写       | 改变的物理量      | 例子                        | 状态数 (M) | 每码元比特数 (n) | 核心要点                                 | 常见应用                 |
| ----- | ---------- | ---------- | ----------- | ------------------------- | ------- | ---------- | ------------------------------------ | -------------------- |
| 基础二进制 | **幅移键控**   | **ASK**    | **振幅**      | **_0: 无信号；1: 最大幅度的信号_**   | **2**   | **1 bit**  | **容易受噪声干扰，类似于“开/关”信号。**              | **光纤通信**                 |
|       | **频移键控**   | **FSK**    | **频率**      | **_0: 900 Hz；1: 901 Hz_** | **2**   | **1 bit**  | ***唯一* 需要两个不同频率载波的二进制方式。**           | **低速无线通信、电话系统**          |
|       | **相移键控**   | **PSK**    | **相位**      | **_0: 0°相位；1: 180°相位_**   | **2**   | **1 bit**  | **抗干扰能力优于 ASK 和 FSK。**               | **高速无线通信，例如 Wi-Fi**      |
|       | 差分相移键控     | DPSK       | 相对相位        |                           | 2       | 1 bit      | 利用前后码元的相位差表示数据。                      |                      |
| 多元/混合 | 正交相移键控     | QPSK       | 相位          |                           | 4       | 2 bit      | 拥有 4 种相位状态（如 45°, 135°, 225°, 315°）。 |                      |
|       | 8PSK       | 8PSK       | 相位          |                           | 8       | 3 bit      | 拥有 8 种相位状态。                          |                      |
|       | **正交振幅调制** | **16-QAM** | **振幅 + 相位** | **_16 种不同的幅度和相位组合_**      | **16**  | **4 bit**  | **混合调制，极大提高频谱利用率。**                  | **数字电视、Wi-Fi、有线和无线通信系统** |
|       | 正交振幅调制     | 64-QAM     | 振幅 + 相位     |                           | 64      | 6 bit      | 常用于高速 WiFi 和 移动网络（如 4G/5G）。          |                      |
