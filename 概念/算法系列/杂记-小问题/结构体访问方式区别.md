> ->语法和.语法的区别
### `.` = 我手上就是这个东西，我直接拆开看部件
```cpp
struct A 
{ 
	int x; 
}; 
struct A a;
a.x;
```
这里 `a` 就在你手上，你直接“当面”拿它的成员。
##### 本质动作：
- 把变量 a 当成结构体，直接从它的内存里取成员偏移处的值。
---
### `->` = 我手上只有地址，我要先指向它，再看部件
```cpp
struct A *p = &a;
p->x;
```
你手上不是结构体本体，而是“它放在哪”的地址。  
所以你要做两步：
1. 找到地址对应的结构体内容（解引用 `*p`）
2. 再访问它的 x 成员（`.`）
##### 本质动作：
- 把地址 p 解引用为结构体，再访问成员
### `p->x == (*p).x`
