### 亲属
| **术语**              | **定义/描述**                | **示例**                     |
| ------------------- | ------------------------ | -------------------------- |
| **祖先 (Ancestor)**   | 从根到该结点路径上的所有结点（不含该结点本身）。 | $K$ 的祖先是 $A, B, E$         |
| **子孙 (Descendant)** | 以该结点为根的子树中的所有结点。         | $B$ 的子孙包括 $E, F, K, L$     |
| **双亲 (Parent)**     | 该结点的直接前驱结点。              | $K$ 的双亲是 $E$               |
| **孩子 (Child)**      | 该结点的直接后继结点。              | $E$ 的孩子是 $K, L$            |
| **兄弟 (Sibling)**    | 具有相同双亲的结点。               | $K$ 与 $L$ 是兄弟              |
| **堂兄弟 (Cousin)**    | 双亲在同一层的结点。               | $G$ 与 $E, F, H, I, J$ 是堂兄弟 |
### 度数相关
| **术语**            | **定义/描述**         | **示例**                   |
| ----------------- | ----------------- | ------------------------ |
| **结点的度 (Degree)** | 结点的**孩子**个数       | 结点 $B$ 的度为 2；$D$ 的度为 3   |
| **树的度**           | 树中所有结点度的最大值。      | 该树的度为 3                  |
| **分支结点**          | 度大于 0 的结点（非终端结点）。 | $A, B, C, D, E$          |
| **叶子结点**          | 度为 0 的结点（终端结点）。   | $K, L, F, G, H, I, J, M$ |
### 层次与高度
| **术语**             | **定义/描述**           | **示例**            |
| ------------------ | ------------------- | ----------------- |
| **结点的层次 (Level)**  | 根为第1层，其孩子为第2层，以此类推。 | $A$ 在第1层，$B$ 在第2层 |
| **结点的深度 (Depth)**  | 结点所在的层次。            | $E$ 的深度为 3        |
| **结点的高度 (Height)** | 以该结点为根的子树的高度。       | $B$ 的高度为 3        |
| **树的高度/深度**        | 树中结点的最大层数。          | 该树的高度为 4          |
### 特性与路径
| **术语**                   | **定义/描述**         | **示例**                               |
| ------------------------ | ----------------- | ------------------------------------ |
| **有序树 (Ordered Tree)**   | 子树从左到右有固定次序，不可互换。 | 若交换 $B$ 下面 $E, F$ 位置，树变了             |
| **无序树 (Unordered Tree)** | 子树位置可互换，不影响树的性质。  | 与有序树相反                               |
| **路径 (Path)**            | 两个结点之间经过的结点序列。    | $A$ 到 $K$ 的路径为 $A \to B \to E \to K$ |
| **路径长度**                 | 路径上“边”的数目。        | $A$ 到 $K$ 的路径长度为 3                   |
