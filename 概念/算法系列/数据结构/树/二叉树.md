### 完全二叉树
%% 除了最后一层，其他层都满；最后一层从左往右连续排列，中间不能有空位 %%
```
        1
      /   \
     2     3
    / \   /
   4  5  6
```
#### 和堆的关系
**堆 = 完全二叉树 + 堆序性质**
- 完全二叉树保证结构
- 堆序保证大小关系
所以所有堆都是完全二叉树 
> [!node] 核心公式：
> 已知节点 i -> $父节点=\dfrac{i-1}{2}$ ，0 节点除外 
> 已知父节点 i -> $右子节点=i*2+2$ ， $左子节点=i*2+1$
### 满二叉树
%% 每一层都填满，没有任何空位的二叉树 %%
```
        1
      /   \
     2     3
    / \   / \
   4  5  6   7
```
> [!node] 核心公式：
> $总节点数 = 2^{层数} - 1$ 
> $叶子节点数=2^{层数-1}$ 
> $非叶子节点数=2^{层数-1}-1$ 
### 二叉搜索树 BST
%% 左子树所有节点 < 根 < 右子树所有节点 %%
```
		8
      /   \
     3     10
    / \      \
   1   6      14
      / \     /
     4   7   13
```
- 但没有对树的空位有限制
#### 平衡二叉搜索树 AVL
%% 左右子树高度差 ≤ 1  ，查找稳定 O(log n) %%
> [!node] 核心公式：
> BF = 高度(左子树) - 高度(右子树)
> BF ∈ {-1, 0, 1}
- 超过 ±1 就不平衡了，需要旋转
##### 右旋（RR）
- 用于 **左-左型失衡**（左子树的左子树插入节点）
```    
	y                x  
   / \              / \  
  x   T3   ->      z   y  
 / \                  / \  
z  T2                T2 T3
```
##### 左旋（ LL）
- 用于 **右-右型失衡**（右子树的右子树插入节点）
```
  y                   x  
 / \                 / \  
T1  x     ->        y   z  
   / \             / \  
  T2  z        d   T1 T2
```
##### 左-右（LR）
- 先左旋再右旋
- 对付 **左子树的右孩子插入** 
##### 右-左（RL）
- 先右旋再左旋
- 对付 **右子树的左孩子插入**
