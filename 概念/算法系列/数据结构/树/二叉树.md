## 常见二叉树
### 完全二叉树
%% 除了最后一层，其他层都满；最后一层从左往右连续排列，中间不能有空位 %%
```
        1
      /   \
     2     3
    / \   /
   4  5  6
```
#### 和堆的关系
**堆 = 完全二叉树 + 堆序性质**
- 完全二叉树保证结构
- 堆序保证大小关系
所以所有堆都是完全二叉树 
> [!node] 核心公式：
> 已知节点 i -> $父节点=\dfrac{i-1}{2}$ ，0 节点除外 
> 已知父节点 i -> $右子节点=i*2+2$ ， $左子节点=i*2+1$

---
### 满二叉树
%% 每一层都填满，没有任何空位的二叉树 %%
```
        1
      /   \
     2     3
    / \   / \
   4  5  6   7
```
> [!node] 核心公式：
> $总节点数 = 2^{层数} - 1$ 
> $叶子节点数=2^{层数-1}$ 
> $非叶子节点数=2^{层数-1}-1$ 

---
### 二叉搜索树 BST
%% 左子树所有节点 < 根 < 右子树所有节点 %%
```
		8
      /   \
     3     10
    / \      \
   1   6      14
      / \     /
     4   7   13
```
- 但没有对树的空位有限制
#### 平衡二叉搜索树 AVL
%% 左右子树高度差 ≤ 1  ，查找稳定 O(log n) %%
> [!node] 核心公式：
> BF = 高度(左子树) - 高度(右子树)
> BF ∈ {-1, 0, 1}
- 超过 ±1 就不平衡了，需要旋转
##### 右旋（RR）
- 用于 **左-左型失衡**（左子树的左子树插入节点）
```    
	y                x  
   / \              / \  
  x   T3   ->      z   y  
 / \                  / \  
z  T2                T2 T3
```
##### 左旋（ LL）
- 用于 **右-右型失衡**（右子树的右子树插入节点）
```
  y                   x  
 / \                 / \  
T1  x     ->        y   z  
   / \             / \  
  T2  z        d   T1 T2
```
##### 左-右（LR）
- 先左旋再右旋
- 对付 **左子树的右孩子插入** 
##### 右-左（RL）
- 先右旋再左旋
- 对付 **右子树的左孩子插入**

### 线索二叉树
#### 核心思想
> 左前右后，提升查找效率
- 若某结点无左子树，则将`lchild`指向其在指定遍历序列中的前驱结点
- 若某结点无右子树，则将`rchild`指向其在指定遍历序列中的后继结点

- `ltag=0`表示`lchild`指向左孩子，`ltag=1`表示`lchild`指向前驱
- `rtag=0`表示`rchild`指向右孩子，`rtag=1`表示`rchild`指向后继

## 二叉树操作
### 遍历
#### 先序
> 根 -> 左 -> 右
![[Pasted image 20260521112540.png|254]]
```c++ fold title=前序
void PreOrder(BiTree T)
{ 
	if(T!=NULL)	{
		visit(T);              //访问根结点
		Preorder(T->lchild);   //递归遍历左子树
		Preorder(T->rchild);   //递归遍历右子树
	}
}
```
#### 中序
> 左 -> 根 -> 右
![[Pasted image 20260521112702.png]]
```c++ fold title=中序
void PreOrder(BiTree T)
{ 
	if(T!=NULL)	{
		Preorder(T->lchild);   //递归遍历左子树
		visit(T);              //访问根结点
		Preorder(T->rchild);   //递归遍历右子树
	}
}
```
#### 后序
```c++ fold title=后序
void PreOrder(BiTree T)
{ 
	if(T!=NULL)	{
		Preorder(T->lchild);   //递归遍历左子树
		Preorder(T->rchild);   //递归遍历右子树
		visit(T);              //访问根结点
	}
}
```
#### 层次遍历
##### 横向遍历 BFS
- 使用数据结构：队列
- 步骤：
	1. 弹出节点
	2. 对于弹出的节点，有左加左，有右加右
```cpp fold title=BFS
void LevelOrder(BiTree T)
{ 
	InitQueue(Q); //初始化辅助队列 
	BiTree p; 
	EnQueue(Q,T); //将根结点入队 
	while(!IsEmpty(Q)){ //队列不空则循环 
		DeQueue(Q,p); //队头结点出队 
		visit(p); //访问出队结点 
		if(p->1child!=NULL) 
			EnQueue(Q,p->1child); //若左孩子不空，则左孩子入队 
		if(p->rchild!=NULL) 
			EnQueue(Q,p->rchild); //若右孩子不空，则右孩子入队 
	} 
}
```
