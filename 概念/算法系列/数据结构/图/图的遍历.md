### BFS（广）
#### 遍历过程
1. 访问起始顶点`v`
2. 从`v`出发，依次访问`v`的所有未被访问过的邻接顶点$w_1,w_2,⋯,w_i$ 
3. 依次访问 $w_1,w_2,⋯,w_i$ 的所有尚未访问过的邻接顶点
4. 重复步骤 2-3
5. 直至图中所有可达顶点都被访问为止
#### 实现
> 非递归
###### 数据结构
1. 队列：用于存储当前层顶点的下一层邻接节点
2. 存储图的结构：[[图的存储]] 
###### 实现算法
```cpp fold title=BFS
bool visited[MAX_VERTEX_NUM]; //访问标记数组 
void BFSTraverse(Graph G){ //对图G进行广度优先遍历 
	for(i=0;i<G.vexnum;++i) 
		visited[i]=FALSE; I/访问标记数组初始化 
	InitQueue(Q); //初始化辅助队列Q 
	for(i=0;i<G.vexnum;++i) //从0号顶点开始遍历 
		if(!visited[i]) //对每个连通分量调用一次BFS() 
			BES(G,i); //若v,未访问过，从v,开始调用BFS()
}
```

```cpp fold title=基于邻接表
void BFS(ALGraph G,int i){ 
	visit(i); //访问初始顶点i 
	visited[i]=TRUE; //对i做已访问标记 
	EnQueue(Q,i); //顶点i入队 
	while(!QueueEmpty(Q)){ 
		DeQueue(Q,v); //队首顶点v出队 
		for(p=G.vertices[v].firstarc;p;p=p->nextarc){ //检测v的所有邻接点 w=p->adjvex; 
			if(visited[w]==FALSE){ 
				visit(w); //w为v的尚未访问的邻接点，访问w 
				visited[w]=TRUE; //对w做已访问标记 
				EnQueue(Q,w); //顶点w入队
			}
		}
	}
} 
```

```cpp fold title=基于邻接矩阵
void BFS(MGraph G,int i){ 
	visit(i); //访问初始顶点i 
	visited[i]=TRUE; //对i做已访问标记 
	EnQueue(Q,i); //顶点i入队 
	while(!QueueEmpty(Q)){ 
		DeQueue(Q,v); //队首顶点v出队 
		for(w=0;w<G.vexnum;w++) //检测v的所有邻接点 
			if(visited[w]==FALSE&&G.edge[v][w]==1){ 
				visit(w); //w为v的尚未访问的邻接点，访问w 
				visited[w]=TRUE; //对w做已访问标记 
				EnQueue(Q,w); //顶点w入队 
			} 
	}
}	

```
#### 性能分析
###### 空间复杂度
- 每个顶点最多入队一次，在最坏的情况下，空间复杂度为$O(V)$ 
###### 时间复杂度 
- 邻接表存储：$O(V+E)$ 
- 邻接矩阵存储：$O(V^2)$ 
#### 最短路径问题
> 可解决非带权图的单源最短路径问题
#### 广度优先生成树
![[Pasted image 20260712150729.png]]
- 邻接矩阵存储：从同一源点出发得到的广度优先生成树唯一
- 邻接表存储：遍历时邻接点的访问顺序不确定，因此生成的广度优先生成树也可能不唯一

### DFS（深）
#### 遍历过程
1. 访问图中的某一起始顶点`v`
2. 从`v`出发，访问`v`的一个未访问邻接顶点 $w_1$ ，再访问 $w_1$ 的一个未访问邻接顶点 $w_2$ ⋯⋯以此类推
3. 当无法继续向下访问时，回退至上 一顶点（递归返回），检查其是否还有未访问的邻接顶点
4. 若有，则从该顶点开始继续上述 2-3 搜索过程
5. 直到图中所有顶点均被访问为止
#### 实现
> 递归
###### 数据结构
1. 存储图的结构：[[图的存储]] 
###### 实现算法 
```cpp fold title=DFS
bool visited[MAX_VERTEX_NUM]; //访问标记数组 
void DFSTraverse(Graph G){ //对图G进行深度优先遍历 
	for(i=0;i<G.vexnum;i++) 
		visited[i]=FALSE; //初始化已访问标记数组 
	for(i=0;i<G.vexnum;i++) //本代码从v?开始遍历 
		if(!visited[i]) //对尚未访问的顶点调用DFS() 
			DES(G,i); 
}
```

```cpp fold title=邻接表
void DFS(ALGraph G,int i){ 
	visit(i); //访问初始顶点i 
	visited[i]=TRUE; //对i做已访问标记 
	for(p=G.vertices[i].firstarc;p;p=p->nextarc){//检测i的所有邻接点 
		j=p->adjvex; 
		if(visited[j]==FALSE) 
			DFS(G,j); //j为i的尚未访问的邻接点，递归访问j
	}
}
```

```cpp fold title=邻接矩阵
void DES(MGraph G,int i){ 
	visit(i); //访问初始顶点i 
	visited[i]=TRUE; //对i做已访问标记 
	for(j=0;j<G.vexnum;j++){ //检测i的所有邻接点 
		if(visited[j]==FALSE&&G.edge[i][j]==1) 
			DFS(G,j); //j为i的尚未访问的邻接点，递归访问j 
	}
}
```
#### 性能分析
###### 空间复杂度
- 最坏情况下， 递归深度可达 $V$ ，因此空间复杂度为 $O(V)$ （递归工作栈所需空间）
###### 时间复杂度
- 邻接表：$O(V+E)$ 
- 邻接矩阵：$O(V^2)$

### 连通性判断
#### 有向图
- 仅当从初始顶点到达图中每个顶点**都存在路径时**，才能通过一次遍历访问到所有顶点
- 否则，将不能访问所有顶点
- 即使整个图是弱连通的，也可能包含强连通分量和非强连通分量
- 在非强连通分量中，单次调用 $BES(G,i)$ 或 $DES(G,i)$ 不一定能访问到该子图的所有顶点
#### 无向图
- 若图是连通的，则只需一次遍历即可访问所有顶点
- 若图是非连通的，则一次遍历仅能覆盖其**所在连通分量**（无向图中的极大连通子图）的所有顶点
> 因此，调用 $BFS(G,i)$ 或 $DFS(G,i)$ 的次数恰好等于该图的连通分量数，因为每次调用都会遍历一个连通分量中的所有顶点
