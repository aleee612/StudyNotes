### 特点
- 访问时间O(1)
- 存储密度高
- 插入需要O(n)
- 每个元素的数据类型相同
- 线性表的下标由1开始，而数组由0开始 
- 顺序存储也称为序号由1开始
### 为什么说顺序表和一维数组在逻辑结构中是不同的

| 对比点    | 一维数组       | 顺序表                |
| ------ | ---------- | ------------------ |
| 结构性质   | 物理结构（内存布局） | 逻辑结构（抽象数据模型）       |
| 是否有长度  | 没有（只有容量）   | 有 length 字段        |
| 是否可扩容  | ❌ 不可扩容     | ✔ 可扩容（动态分配）        |
| 存储位置   | 栈 or 静态区   | 通常堆分配              |
| 有无操作封装 | ❌ 没有操作     | ✔ 有插入删除查找等接口       |
| 生命周期   | 编译器控制      | 程序员控制（malloc/free） |
| 表示范围   | 容量固定       | 可以根据逻辑需要扩展         |
