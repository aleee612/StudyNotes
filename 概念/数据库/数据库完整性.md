### 实体完整性：
##### 定义：(主码)
```sql
……
Sno CHAR(20) PRIMARY KEY, /*在列级定义主码，仅单个*/
……
PRIMARY KEY(Sno，Cno) /*在表级定义主码，多个必须表级*/
……
```
### 参照完整性：
##### 定义：（外码）
```sql
……
FOREIGN KEY(Sno) REFERENCES Student(Sno), /*多个定义写多行，Sno为外码，参照表是Student*/
……
```
### 属性约束（皆在创建表格时）：
##### 定义：
```sql
#列值非空
Sno CHAR(8) NOT NULL,
#列值唯一
SHname VARCHAR(40) UNIQUE,
#检查一个列值是否满足条件
Ssex CHAR(6) CHECK(Ssex IN('男','女')),
```
### 元组约束：

##### 定义：
```sql
CHECK(Ssex='女' OR Sname NOT LIKE 'Ms.%')
```
### 完整性约束：
##### 定义：
```sql
CONSTRAINT <完整性约束名> <完整性约束>
#e.g:
Sno CHAR(8)
CONSTRAINT C1 CHECK(Sno BETWEEN '10000000' AND '29999999'),
```
##### 修改：
一般常用ADD，DROP
```sql
ALTER TABLE Student
DROP CONSTRAINT C3;
```
##### 域：
```sql
#定义：
CREATE DOMAIN GenderDomain CHAR(6)
CHECK(VALUE IN ('男','女'));

CREATE DOMAIN GenderDomain CHAR(6)
CONSTRAINT CD CHECK(VALUE IN ('男','女'));
#修改：
ALTER DOMAIN GenderDomain
DROP CONSTRAINT CD;
```
### 触发器：
##### 定义：
- REFERENCING在mysql中无法使用，直接使用NEW.column_name或OLD.column_name
- mysql无FOE EACH STATEMENT
```sql
DELIMITER//
CREATE TRIGGER <触发器名>
|BEFORE | AFTER| <触发事件> ON <表名>

REFERENCING NEW | OLD AS <变量>
FOR EACH |ROW | STATEMENT|
[ WHEN <触发条件> ] <触发动作体>
//
DELIMITER
```
##### 删除：
```sql
DROP TRIGGER <触发器名> ON <表名>
```
