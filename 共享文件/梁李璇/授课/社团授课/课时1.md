# 开发环境

C++ 需要 编写代码 → 编译 → 运行，是**编译型语言**
不像 Python 那样可以直接运行，需要经过以下步骤：
1. 编写代码（.cpp 文件）
2. 编译（Compile）：将 C++ 代码转换为二进制代码
3. 链接（Link）：将多个 .o（目标文件）和库文件合并，生成可执行文件（.exe）
4. 运行：执行最终生成的可执行文件
---
### C++ 代码的编译器：
1. GCC（GNU Compiler Collection）：开源，Linux/macOS 默认支持，Windows 需安装 MinGW
2. Clang：苹果 macOS 自带，性能优越
3. MSVC（Microsoft Visual C++）：Windows 下的官方编译器，随 Visual Studio 一起安装
- 如何检查编译器版本？
```
g++ --version  # 检查 GCC 版本
clang++ --version  # 检查 Clang 版本
cl.exe  # 检查 MSVC 版本
```

# 命名空间（namespace）
- 为什么需要命名空间？
在 C++ 中，不同的代码库可能会定义相同名称的变量、函数或类
如果在另一个库或文件中也定义了相同名称的函数，就会导致**命名冲突**。
C 语言里通常用 **前缀** 解决这个问题，但这种方式不够优雅。C++ 引入了 命名空间（namespace） 来解决这个问题。

---

- 命名空间的基本用法
```cpp
namespace 空间名 {
// 变量、函数、类等
}
```
示例：
```cpp
#include <iostream>

// 定义命名空间 A
namespace A {
void printMessage() {
std::cout << "Hello from A" << std::endl;
}
}
// 定义命名空间 B
namespace B {
void printMessage() {
std::cout << "Hello from B" << std::endl;
}
}

int main() {
A::printMessage();  // 调用 A 命名空间中的函数
B::printMessage();  // 调用 B 命名空间中的函数
return 0;
}
```
- 使用 using 关键字
如果 A:: 或 B:: 这样的前缀太麻烦，可以使用 using 关键字。
```cpp
#include <iostream>

namespace A {
void show() {
        return 1;
}
}

using namespace A;  // 之后代码可以直接调用 A 的内容

int main() {
show();  // 相当于 A::show()
return 0;
}
```
- 使用 using 声明单个成员
如果只想引入命名空间里的某个特定成员：
```cpp
#include <iostream>

namespace A {
void show() { return 1; }
void hide() { return 1; }
}

using A::show;  // 只引入 show()，不引入 hide()

int main() {
show();  // 直接调用 A::show()
// hide(); // 错误：未引入 hide()
return 0;
}
```
## std命名空间
- `std` 是标准库（Standard Library）的命名空间，包含了 C++ 标准库中几乎所有的功能，比如输入输出、[[容器]]、算法等
- **常见的** `std` **组件**

|   |   |
|---|---|
|组件|作用|
|`std::cout, std::cin, std::cerr`|标准输入输出|
|`std::string`|字符串类|
|`std::vector, std::map, std::set`|容器|
|`std::sort, std::find, std::for_each`|算法|
|`std::thread, std::mutex`|线程库|
|`std::shared_ptr, std::unique_ptr`|智能指针|

```cpp
#include <iostream>
using namespace std;
int main() {
    cout << "hello world"; 
    return 0;
}
```
# 头文件
- cpp的头文件和c作用相似，都是为了防止**重复定义**
- 在c 中我们知道有常用的头文件，比如stdio.h(c中的标准输入输出库)，math.h（数学库）
- 在cpp中我们都保留了这些库，只不过头文件换了个名称cstdio,cmath
- 那么cpp中有哪些头文件呢？
### 常用的头文件：
- `<iostream>` 及其相关头文件是 C++ 标准库中 **输入输出（I/O）流** 相关的库，提供了 **标准输入（**`cin`**）**、**标准输出（**`cout`**）** 和 **标准错误（**`cerr`**）** 等功能。
- `<algorithm>` 提供了多种常用的算法功能 如排序、查找、替换、排序、集合运算 这个库提供了基于迭代器的算法，使得它们不仅可以用于标准[[容器]]（如 `vector`, `list` 等），还可以应用于自定义[[容器]]和数据结构。
- `#include<bits/stdc++.h>`

# 输入输出（I/O）
### iostream 基本输入输出
#### 头文件：`#include <iostream>`
- 基本输出：`std::cout`
- cout (character output) 用于向标准输出（屏幕）打印信息：
- `std::endl` 的作用：换行（等同于 \n）
- 可以链式输出：
```
#include <iostream>  // 引入头文件

int main() {
std::cout << "Hello, C++!" << std::endl;
return 0;
}
```

```
std::cout << "My age is " << 20 << " years old." << std::endl;
```

---

基本输入：`std::cin`

cin（character input）用于从标准输入（键盘）读取数据：

- 多个变量输入

- cpp的输入和c一样，是不能包含换行符的

```
#include <iostream>

int main() {
int age;
std::cout << "Enter your age: ";
std::cin >> age;  // 读取输入并存入变量 age
std::cout << "You are " << age << " years old." << std::endl;
return 0;
}
```

```
int a, b;
std::cin >> a >> b;  // 依次输入两个整数
std::cout << "Sum: " << (a + b) << std::endl;
```

```
std::cin >> a >> b >> endl;//报错
```

### 输入输出格式化

- c风格输出：

```
printf("%.2f",a);
```

- C++ 提供 iomanip 头文件来控制输出格式：

```
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
double pi = 3.14159265358979;

cout << fixed << setprecision(2);
cout << "Pi: " << pi << endl;  // 输出：Pi: 3.14

return 0;
}
```

- 常见格式化函数

|   |   |
|---|---|
|函数|作用|
|`std::setw(n)`|设定宽度（右对齐）|
|`std::setfill(c)`|设置填充字符|
|`std::fixed`|以固定小数点格式显示浮点数|
|`std::scientific`|以科学计数法显示浮点数|
|`std::setprecision(n)`|设置小数位数|

示例：

```
std::cout << std::setw(10) << 42 << std::endl;  // 右对齐，宽度 10
std::cout << std::setfill('*') << std::setw(10) << 42 << std::endl; // 用 * 填充
```
