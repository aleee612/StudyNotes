- 无论哪种循环写法，我们注重的都是：何时开始循环？何时终止循环？
---
## 基本语法
### while
`while` 先判断条件，再执行循环体。  
适用于：**次数不确定、需要先判断再执行的场景**。
**语法：**
```c
while (条件) {
    // 循环体
}
```
**特点：**
- 条件为真 → 执行循环体
- 条件为假 → 立即结束循环
- 可能一次都不执行
---
### do-while
`do-while` **先执行一次循环体，再判断条件**。  
适用于：**至少要执行一次的情况**。
**语法：**
```c
do {
    // 循环体
} while (条件);
```
**特点：**
- 不管条件是否成立，**循环体至少执行一次**
- 后判断
---
### for
`for` 循环适用于**已知次数**的场景，结构清晰，把初始化、判断、更新写在同一行。
**语法：**
```c
for (初始化; 条件; 更新) {
    // 循环体
}
for(int i = 0; i < 100; i++){
	//
}
```
**特点：**
- 常用于计数循环，如遍历数组等
---
## 终止循环
### break
`break` 用于**立即跳出最近的一层循环**。
**作用：**
- 立刻终止循环，不再继续执行任何循环内部代码
- 常用于找到目标立即退出，如搜索、菜单选择等
**示例：**
```c
for(;;){
	for (int i = 0; i < 10; i++) {
	    if (i == 5) break;
	}
}
```
---
### continue
`continue` 用于**跳过本次循环剩余语句，直接进入下一次循环判断**。
**作用：**
- 不结束循环，只是提前进入下一轮
- 常用于跳过特定条件的处理
**示例：**
```c
for (int i = 0; i < 10; i++) {
	...
    if (i % 2 == 0) continue; // 跳过偶数
    printf("%d ", i);
    ...
}
```
