# 矩阵与向量

### 表述方法
```matlab
a = [1,2,3] % 行向量
a = [1 2 3]
a = [1;2;3] % 列向量
E = [1 2;3 4;5 6] % 三行两列矩阵
```

### 查询
```matlab
E = [1 2 3;4 5 6;7 8 9]
E(2,1) % 查询4
E(2,:) % 取第二行全部
E([1,2],:) % 取第一三行
E(1:3,:) % 取第一到第三行
E(1:2:3,:) % 取第一行和第三行，每次递增两行，到第三行结束
E(1:end,:) % 取第一行到最后一行
E(:,1) % 取第一列全部
E(:) % 取全部
```

# 函数
## sum求和

- 如果是向量（无论是行向量还是列向量）都是直接求和
```matlab
E = [1 2 3]
sum(E)
E = [1;2;3]
sum(E)
```

- 如果是矩阵，则需要说明
```matlab
E = [1 2;3 4;5 6]
a = sum(E) % 默认列求和
a = sum(E,1) % 列求和得行向量
a = sum(E,2) % 行求和得列向量
% 整个求和
a = sum(sum((E))
a = sum(E(:))
```

^4b0668

## size函数
- 求矩阵大小，返回一个向量，第一个元素是矩阵的行数，第二个元素是矩阵的列数
```matlab
E = [1 2 3;4 5 6]
(r,c) = size(E)
r = size(E,1) % 求行数
r = size(E,2) % 求列数
```

## repmat函数
- `B = repmat(A,m,n)` 将矩阵A复制m*n块，即把A作为B的元素，B由m*n个A平铺而成（就是复制粘贴） ^5f82fc
```matlab
A = [1,2,3;4,5,6]
B = repmat(A,2,1)
```

# 矩阵的运算

### 乘法
```matlab
A = [1 2;3 4]
B = [5 6;7 8]
A * B
A .* B % 形状相同的矩阵对应的元素之间的乘法，即点乘
A * 2 % 每个元素同时乘相同的倍数
A .* 2
A ^ 2 % 等同于A * A
A .^ 2 % A中每个元素平方
```

### 逆矩阵
```matlab
inv(B)
```
### 除法
```matlab
A / B
A ./ B % 点除
A / 2 % 每个元素同时除相同的倍数
A ./ 2
```

# 特征值和特征向量

- 求A的全部特征值，返回一个由特征值构成的向量
```matlab
E = eig(A)
```

- 对A的全部特征值，构成对角阵D，并求A的特征向量构成V的列向量（V的每一列都是D与之相同列的特征值的特征向量）
```matlab
[V,D] = eig(A)
```
