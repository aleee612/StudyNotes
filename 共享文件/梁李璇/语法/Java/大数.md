### 存放位置：java.math
##### 导入
``` java
import java.math.*;
```
### BigInteger类与BigDecimal类
- 可以处理包含任意长度数字序列的[[语法/Java/数组|数组]] BigInteger可以实现任意精度的整数运算，BigDecimal则是浮点数
##### 1. 转换方法
 a. valueOf方法可以静态地将普通数值转换为大数
```java
BigInteger a = BigInteger.valueOf(100);
```
 b. 使用字符串
 ```java
 BigInteger a = new BigInteger("123456789987654321123456789");
 ```
##### 2. 运算方式
- 无法使用+`*`等常规运算符
 a. add方法，对应+
 ```java
 BigInteger c = a.add(b);//c=a+b
 a = a.add(b);//a+=b
 ```
 b. multyply方法，对应`*`
 ```java
 BigInteger d = c.multyply(b.add(BigInteger.valueOf(2)));//d=c*(b+2)
 ```
 - 注： 与cpp不同，Java中没有提供运算符重载功能，因此无法重新定义+和`*`
 - 普通数值需要先用 `BigInteger.valueOf()` 转换
