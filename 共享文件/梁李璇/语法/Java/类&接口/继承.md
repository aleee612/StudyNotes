### 类，超类和子类
- 所有 Java 对象都继承自 `[[Object]]` 类
#### 定义子类
```java
public class Mng extends Emp
{
	...
}
```
- 在Java中所有的继承都是公共继承
- Emp：超类 / 基类 / 父类
- Mng：子类 / 派生类
#### 覆盖方法（重写）
- 子类中可以使用`@override`来标明这是一个重写函数，若函数名在父类中没有出现则报错
```java
@override
public double getSalary()
{
	...
}
```
- 子类的方法无法直接使用父类的字段，若想访问需要使用接口
- 使用父类方法而非自身同名方法：
```java
public double getSalary()
{
	double bS = super.getSalary();
	return bS + this.bonus;
}
```
- super与this不同于：super不是一个对象的引用，而不能将super赋给另一个对象变量
- 在覆盖一个方法时需要保证[[返回类型的兼容性]]
- 子类方法不能低于超类方法的[[访问说明符|方法可见性]]
#### 子类构造器
```java
public Mng(String n)
{
	super(n)
	this.bonus = 0;
}
```
- 这里的super是指调用父类中带有n参数的构造器
- 使用super调用构造器的语句必须是子类构造器中的第一条语句
- 若无显示指定父类构造器，则默认调用父类中无参数的构造器；若超类中也没有无参构造器则报错
