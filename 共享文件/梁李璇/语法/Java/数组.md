- 在Java中，可以视为所有的数组名其实都只是一个指针，只是对于数组的一个引用
- 可以理解为
```java
int[] a = new int[100];
```
就相当于c++中的：
```cpp
int* a = new int[100];
```
## 数组的声明
1.  `[]` 跟在数组变量（名）之后
```java
char a[];
char b[][];
```
2. `[]` 跟在数组类型之后，可以同时声名多个（用的多）
```java
char[] a,b,c;
int[][] d,e;
```
## 数组的创建/初始化
- Java允许在运行时确定数组的大小
### 创建&初始化
1. 使用new进行初始化
```java
s = new char[20];
int[] arr = new int[]{1, 2, 3};
int[][] arr1 = new int[3][4];
```
2. 和c/cpp相似的在声明同时初始化
```java
int[] arr = {1, 2, 3};
int[][] arr = {{1,2,3},{4,5,6}};
```
- Java 的二维数组 **本质是数组的数组**，所以每行可以长度不一样：
3. 赋值一个匿名数组
```java
arr = new int[] {1,2,3,4,5};
```
- 这样可以重新初始化一个数组而无需创建新的变量
3. 二维的初始化
```java
int[][] arr2 = {{1, 2},
			{3, 4, 5}}; 
System.out.println(arr2[1][2]); // 输出 5
```
- 这就是所谓 **不规则二维数组**（Jagged Array）。
### 数组长度
```java
int len = arr.length;
```
- 数组的长度定义后无法更改，但可以将数组变量的指向换一个对象
### 对象数组
- 可以理解为是和c语言的数据结构中的struct的数组
## 快速打印
- 调用Array.toString(a)，返回一个包含数组元素的字符串`{1,2,3,4}`
```java
System.out.println(Arrays.toString(a));
```
## 数组排序
1. 导入头文件
```java
import java.util.*;
```
2. 使用sort排序
```java
int[] a = new int[10000];
...
Arrays.sort(a);
```
