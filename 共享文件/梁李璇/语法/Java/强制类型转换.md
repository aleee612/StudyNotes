### 转换方式
- 使用圆括号括起来
```java
double x = 3.405;
int nx = (int) x;
```
- 大数类的转换：[[大数#1. 转换方法]]
### 查看是否成功转换：[[instanceof]]
```java
if(staff[1] instanceof Manager){
	boss = (Manager) staff[1];
	...
}
```
- 若staff[1]为null，不会返回异常，是返回false
### 注意事项
- 向上转型：永远安全
- 向下转型：只有传的类型相同才安全
```java
public class text {  
    public static void main(String[] arg){  
        ttt a = new ttt("a");  
        tt[] arr = new tt[2];//子数组  
        arr[0] = new tt("zi");  
        arr[1] = new tt("lei");  
        ttt boom = (ttt)arr[0];//子给父，此时父为子的数据  
        arr[1] = (tt) boom;//父给子，但由于父为子的数据，所以赋值成功  
        System.out.println(a.getName());//输出父  
        System.out.println(arr[1].getName());//输出0（子）传a（父）传1（子）的（0（子）的）数据  
        a = arr[1];  
        System.out.println(a.getName());//输出转为子后的父  
    }  
}  
class ttt{  
    private String name = "error";  
    public ttt(String n){  
        name = n;  
    }  
    public String getName(){  
        return name;  
    }  
}  
class tt extends ttt{//子  
    public tt(String n){  
        super(n);  
    }  
}
```
