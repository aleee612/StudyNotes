## 参数类型
- 位置参数，默认值参数，关键字参数，不定长参数
### 位置参数：
- 实参形参一一对应
### 默认值参数：
- 创建时就有，不写形参没关系，不报错。默认值参数必须出现在函数形参列表的最右端。
```python
def funname(...,形参名 = 默认值):
        函数体
```
- 查看函数所有默认值参数的当前值：`funname.__defaults__` ，返回一个元组，依次表示每个默认值参数的当前值。
- 要避免使用列表，字典，集合等可变数据类型作为默认值参数。
### 关键字参数：
按参数名传递，允许顺序不一致
```python
def fun(say,name):
        pass
fun(say = '',name = '')
```
### 不定长参数：
```python
def fun([形参列表,]*arges,**kwargs):
        pass
```
- `*`：存放在元组中
- `**`：存放在字典中
## 嵌套
- python允许在函数内定义另一个函数
## global与nonlocal
- `global`: 变量已经在函数外创建，需声明为全局变量（global name）
- 若并未创建，则在声明为全局变量后可在函数外使用、修改该变量
- `nonlocal` ：在一个嵌套的函数内使用或修改嵌套外部作用域中的变量
## 匿名函数
```python
fun = lambda [参数列表]:表达式

e.g:
sum = lambda arg1, arg2: arg1+arg2 # 返回arg1+arg2
```
