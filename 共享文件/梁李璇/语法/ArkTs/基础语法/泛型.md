# 用途：

起到占位置的作用，可以传入任意类型的参数，通过泛型，函数或类能够处理多种类型，而无需在每次使用时都进行类型声明。

# 基本语法：

```
function len<T>(list: T[]):number{
  ...[函数体]
}
```

# 多个泛型参数：

```
function pair<T, U>(first: T, second: U): [T, U] {
  return [first, second];
}

let result = pair("hello", 42);  // T 被推导为 string，U 被推导为 number
```

# 泛型约束：

- 可以为泛型类型添加约束，使得它只能接受特定的类型或类型层次。

```
function lengthOf<T extends { length: number }>(item: T): number {
  return item.length;
}

let result = lengthOf("Hello");  // 返回 5
let result2 = lengthOf([1, 2, 3]);  // 返回 3
// lengthOf(42);  // 错误：number 类型没有 length 属性
```

这里的 `T extends { length: number }` 约束了 `T` 必须是具有 `length` 属性的类型。这使得我们能够传入字符串、数组等类型，确保泛型类型在运行时有特定的属性。

# 默认类型：

- 为泛型参数提供默认类型。当没有明确传入类型参数时，使用默认类型。

```
function log<T = string>(value: T): T {
  console.log(value);
  return value;
}

log("Hello");  // T 推导为 string
log(42);  // T 推导为 number
```
