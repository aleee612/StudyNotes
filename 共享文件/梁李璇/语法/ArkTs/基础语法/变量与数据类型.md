```
let Name:string = "aleee";
```

- 全局变量

- 局部变量

# 数据类型

### 布尔类型

```
let isDone : boolean = true;
```

### 数字

- ArkTS中所有数字都是浮点数，所有类型都是number，支持多种进制

```
let literal : number = 6;
```

### 字符串

- 可用单引号也可用双引号，用反引号`` ` ``定义多行文本和内嵌表达式
- 可以使用`+`拼接字符串

```
let str : string = "Aleee";
let ss : string = `你好，${str}!`;//内嵌表达式
let msg : string = "Hello" + " Aleee";//字符串运算符
msg = "mi" + 2025 + "Years"//可拼接数字
//数字 + 字符串 -> 字符串
```

### 数组

- 打印时通过下标获取元素（下标从0开始）
- 或可以使用`JSON.stringfy()`输出全部

```
let list : number[] = [1, 2, 3];//元素类型[]
let arr : Array<number> = [1, 2];//Array<元素类型>(泛型)
console.log(JSON.stringfy(arr));//输出全部
```

### 元组

```
let x : [string, number];
x = ['Hi',1];//必须按声明赋值
```

### 枚举类型

```
enum Color{Whilte = 0xFF,Grey = 0x7F};
let c : Color = Color.White;
```

### 任意类型与联合类型

```
let noSure : any = 'Hi';//任意类型，ets不支持，ts支持，动态改变数据类型
let firstNumber : string | number;//联合类型，‘|’表示‘或’，可以在两种类型之间改动
```

### void & null

- null和Undefined是其他任意类型的子类型，在ArkTS中不支持

```
function hello():void{//说明该方法没有返回值
}
let n : number = 1;
n = null;
n = undefined;
```

### 对象类型

- 一种复合的数据类型

```
interface Person{
  name : string,
  age : number,
  isDone : boolean
}//ArkTS中要先定义类型后才可使用
let person = {
  name = 'Aleee',
  age = 20,
  isDone = true;
}
person.name//访问元素
```

- 一个变量/常量声明包含了初始值，那么开发者就不需要显式地标明类型，函数中也可以进行类型推断

```

```
