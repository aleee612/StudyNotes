# 封装

- 用访问修饰符 `public` / `private` / `protected` 来控制**属性和方法的访问权限**

- 常通过`private` 成员 + `public` 方法来对外提供接口（**读/写数据**）
- 特点（优点）：提供可复用性，安全性

```
class Person {
  private age: number;

  constructor(age: number) {
    this.age = age;
  }

  // 提供访问接口
  getAge(): number {
    return this.age;
  }

  setAge(age: number): void {
    if (age > 0) {
      this.age = age;
    }
  }
}
```

# 继承

- 使用 `extends` 关键字实现继承

- 子类会继承父类的属性和方法

- 子类可以重写（override）父类方法
- arkts只支持单继承（多继承可以通过接口实现）

```
class Animal {
  move(): void {
    console.log("Animal is moving");
  }
}

class Dog extends Animal {
  bark(): void {
    console.log("Woof!");
  }

  // 重写父类方法
  move(): void {
    console.log("Dog runs");
  }
}
```

# 多态

- 同一个方法名在不同类中表现出**不同的行为**

- 多态通常通过**继承 + 方法重写**实现
- 特点（优点）：灵活性，提高代码复用性，提高可读性和可维护性

```
class Animal {
  speak(): void {
    console.log("Animal makes a sound");
  }
}

class Cat extends Animal {
  speak(): void {
    console.log("Meow");
  }
}

class Dog extends Animal {
  speak(): void {
    console.log("Woof");
  }
}

function makeAnimalSpeak(animal: Animal): void {
  animal.speak(); // 实际调用哪个方法，取决于传入的对象
}

makeAnimalSpeak(new Dog()); // Woof
makeAnimalSpeak(new Cat()); // Meow
```
