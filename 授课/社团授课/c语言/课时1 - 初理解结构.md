##### 什么是变量？什么是数据类型？什么是语句？什么是IO？
| 类型                   | 说明      | `printf` 用法    | `scanf` 用法 | 备注                                            |
| -------------------- | ------- | -------------- | ---------- | --------------------------------------------- |
| `int`                | 整数      | `%d`           | `%d`       | 标准整型                                          |
| `long`               | 长整型     | `%ld`          | `%ld`      | 比 `int` 大（通常 8字节）                             |
| `short`              | 短整型     | `%hd`          | `%hd`      | 比 `int` 小（通常 2字节）                             |
| `float`              | 单精度浮点   | `%f`           | `%f`       | 注意：`printf` 中 `float` 会被自动提升为 `double`        |
| `double`             | 双精度浮点   | `%f` （或 `%lf`） | `%lf`      | `printf` 中 `%f` 和 `%lf` 等价，但 `scanf` 必须 `%lf` |
| `char`               | 单个字符    | `%c`           | `%c`       | `char` 实际是1字节整数                               |
| `char[]`             | 字符串     | `%s`           | `%s`       | 不用加 `&`                                       |
| `_Bool` / `bool`     | 布尔值     | `%d`           | `%d`       | C 没有原生 `%b`，打印时用 `%d` 显示为 0/1                 |
| `unsigned int`       | 无符号整数   | `%u`           | `%u`       | 永远非负                                          |
| `long long`          | 超长整型    | `%lld`         | `%lld`     | 常用于大数计算                                       |
| `unsigned long long` | 无符号超长整型 | `%llu`         | `%llu`     | 常用于正大数                                        |
```c

##### 类型转换这一块
| 语法示例                | 作用                    | 说明                 |
| ------------------- | --------------------- | ------------------ |
| `(int) 3.14`        | 将 `3.14` 转成整数         | 小数部分被舍去，结果为 `3`    |
| `(float) 5`         | 将整数 `5` 转成浮点数         | 结果为 `5.0`          |
| `(double) 2.7f`     | 将 `float` 转成 `double` | 精度提升               |
| `(char) 65`         | 将整数 `65` 转成字符         | 对应 ASCII，结果为 `'A'` |
| `(unsigned int) -1` | 将负数转为无符号整数            | 结果为最大值（取模）         |
| `(short) 1000`      | 将整数转成 `short`         | 超过范围可能溢出           |
##### 那这时候就有人要问了，老师老师，我每输出一次hello world就要打一次hello world，有没有什么不吃时间和复杂度的写法呢？
- 有的兄弟有的

int main() {
    char str[] = "hello world";

    return 0;
}
```
##### 如何输出自己输入的东西呢？
```c
#include<stdio.h>
int HelloWorld3() {
    int num;
    scanf_s("%d", &num);
    printf("num = %d\n", num);
    return 0;
}
```
##### 判断语句大家都知道，那么在做判断语句的时候又有什么注意要点呢？
```c
#include<stdio.h>
int HelloWorld3() {
    int num;
    scanf_s("%d", &num);
    if (num > 0){
	    printf("num是正数");
    }
    else if(num == 0){
	    printf("num是零");
    }
    else{
	    printf("num是负数");
    }
    printf("num = %d\n", num);
    return 0;
}
```
