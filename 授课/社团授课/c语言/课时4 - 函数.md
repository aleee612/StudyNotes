### 函数结构 & 特点
1. **把一段可重复使用的代码打包**  
    ——减少重复劳动，提高代码复用率。
2. **可以接收参数**  
    ——让同一套逻辑处理不同的数据。
3. **可以返回一个结果**  
    ——把处理后的值带回去给调用者。
4. **必须先定义（声明）再使用**  
    ——编译器要提前知道函数长啥样。
5. **让程序结构更清晰**  
    ——逻辑拆分，各司其职，像组件一样。

### 函数的定义与使用
```c
float add(int a, int b) {  //形式参数 void->不需要返回值
	float c = a + b;
	for(;;){
		 ...
		 for(;;){
			 ...
			 //跳出
			 //return;
		 }
	}
	return c; 
}  
int main() {
	int sum = add(3, 4);  //实际参数
	printf("%d\n", sum);     
	return 0; 
}
```
---
### return 的特点
1. **return 立刻结束函数**
2. **返回值类型必须和函数声明一致**
3. **void 函数可以不写 return**，写了 return; 表示提前结束，但不能返回值。
---
### 函数声明（原型）
当函数写在 `main()` 后时，编译器会不知道它的存在，需要提前声明：
- 所以这也就说明一点，在c中定义在什么位置是非常重要的
```c
int add(int a, int b);  // 声明  
int main(){     
	printf("%d\n", add(2,3));     
	return 0; 
}  
int add(int a, int b){  // 定义     
	return a + b; 
}
```

### 参数传递（重点：值传递）
C 语言的函数参数 **全部都是“值传递”**（copy 一份过去）。
例子：
```c
void change(int x){
     x = 100; 
}  
int main(){     
	int a = 5;     
	change(a);     
	printf("%d\n", a);  // 还是 5     
	return 0; 
}
```
- a 不会变，因为你只是把 **a 的值复制了一份给 x**。

### 函数与数组
数组作为参数时其实传的是**数组首地址**。
```c
void printArr(int arr[],int len){
	for(int i=0; i<len; i++){         
		arr[i] +=1 ;
		printf("%d ", arr[i]);     
	} 
}  
int main(){     
	int a[] = {1,2,3,4,5};     
	printArr(a, 5);  // 只传数组名即可     
	return 0; 
}
```
- 注意事项：
	- [[sizeof为什么获取不了arr的长度？]] 
	- 二维数组作为函数参数
```c
void print2D(int arr[][3], int rows){     
	for(int i=0;i<rows;i++){         
		for(int j=0;j<3;j++){             
			printf("%d ", arr[i][j]);         
		}         
	printf("\n");     
	} 
}  
int main(){     
	int a[2][3] = {{1,2,3},{4,5,6}};     
	print2D(a, 2); 
}
		
```
- 注意：第二维必须写死。
