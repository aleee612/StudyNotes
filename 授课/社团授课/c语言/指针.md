### 1. 指针的概念

| **特点** | **描述**                                                         |
| ------ | -------------------------------------------------------------- |
| **定义** | 指针是一个**变量**，它存储的值是另一个变量的**内存地址**。                              |
| **类型** | 指针变量也有类型，它表明该指针指向的内存区域存储的是什么类型的数据（如 `int*`, `char*`）。          |
| **大小** | 在 32 位系统上通常是 4 字节，在 64 位系统上通常是 8 字节。指针的大小与它指向的数据类型无关，只与系统架构有关。 |

### 2. 指针的声明与初始化
声明指针时，使用星号 `*` 来表示该变量是一个指针。
```c
int num = 10; 
int *p;         // 声明一个指向 int 类型的指针 p
p = &num;       // 初始化：将变量 num 的地址赋值给指针 p
```
### 3. 指针的核心操作符

|**操作符**|**名称**|**用途**|**示例**|**结果**|
|---|---|---|---|---|
|`&`|**取地址符** (Address-of)|获取变量的内存地址。|`&num`|`num` 变量在内存中的地址 (例如：`0x7ffeefbff604`)|
|`*`|**解引用符** (Dereference)|访问指针所指向地址上的值。|`*p`|指针 `p` 所指向内存地址中的值，即 `10`|

#### 内存地址与解引用示意
```c
int num = 10;
int *p = &num;

printf("num 的值: %d\n", num);         // 10
printf("num 的地址: %p\n", &num);       // 0x...A00
printf("p 中存储的值 (即 num 的地址): %p\n", p); // 0x...A00
printf("p 指向的值 (*p): %d\n", *p);     // 10

*p = 25; // 通过指针修改变量 num 的值
printf("num 的新值: %d\n", num);         // 25
```

### 4. 指针与函数参数（值传递的突破）
C 语言函数参数传递是**值传递**。通过传递指针（即传递变量的地址），可以实现在函数内部修改函数外部的变量，这是指针最核心的应用之一。

|**传递方式**|**作用**|**示例函数签名**|
|---|---|---|
|**值传递** (Pass by Value)|只能修改传入的副本。|`void func(int x)`|
|**指针传递** (Pass by Pointer)|传入变量的地址，通过解引用修改**原始变量**。|`void func(int *p)`|

#### 示例：使用指针修改变量
```c
// 接收一个整型指针 p
void modify_value(int *p) {
    *p = 100; // 解引用 p，将它指向的内存地址中的值改为 100
}

int main() {
    int data = 50;
    printf("修改前 data: %d\n", data); // 50
    
    // 传入 data 的地址
    modify_value(&data); 
    
    printf("修改后 data: %d\n", data); // 100
    return 0;
}
```

### 5. 指针与数组
在 C 语言中，数组名在大多数情况下可以被视为指向其**首元素**的**常量指针**。
```c
int arr[5] = {10, 20, 30, 40, 50};

// 数组名 arr 就是首元素的地址
printf("数组首元素地址 (arr): %p\n", arr);       // 0x...A00
printf("数组首元素地址 (&arr[0]): %p\n", &arr[0]); // 0x...A00

int *p = arr; // 将数组名赋值给指针 p
printf("指针 p 指向的值 (*p): %d\n", *p);        // 10
```
#### 指针的算术运算 (Pointer Arithmetic)
指针可以进行加减运算。指针加 1 意味着地址增加**一个其所指向数据类型的大小**。
- 对于 `int *p`：`p + 1` 会指向下一个 `int` 元素。地址会增加 `sizeof(int)` 字节。
```c
// 承接上例：p 指向 arr[0] (值为 10)
printf("p+1 指向的值: %d\n", *(p + 1)); // 20 (即 arr[1])

// 数组的下标操作符 [] 本质上也是指针算术的语法糖：
// arr[i] 等价于 *(arr + i)
```
