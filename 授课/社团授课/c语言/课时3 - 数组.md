### 数组结构 & 特点
1. 顺序存储：使用连续的内存空间来存储数据
2. 存储的数据为同一种数据类型（内存大小空间相同 -> 可下标索引）
3. 长度固定
4. 改、查速度快
5. 下标访问：0开始，length-1结束
### 数组的定义和操作
#### 定义 
```c
int a[100]; //空数组
int b[5] = {1，2，3，4，5}; //直接定义数组
int c[5] = {1,2}; //定义一部分的值剩余的（从第三个元素开始）为0
int d[] = {1,2,3,4}; //不写长度，但赋值，由编译器计算数组长度
int n;//但是n什么都不写，没有赋值，没有输入（事实上输入也会出错，因为编译器编译阶段无法识别数组到底有多长（总得先编译运行再输入对吧））
int arr[n];//就会报错
```
#### 增删改查
- 操作起来和普通的变量无异
```c
a[0] = 1;//给第一个元素赋值
a[0] = 2;
scanf("%d",&a[1]);//输入第二个元素
printf("%d",a[2]);//输出第三个元素
```
### 通过遍历操作数组
```c
for(int i = 0;i<10;i++){
	printf("%d",a[i]);
}
```
### 扩展
1. 字符串的定义
```c
#include <stdio.h>

int main() {
    char s1[] = "hello";       // 自动含 '\0'，长度 6
    char s2[20] = "hi";        // 剩余位置自动补 '\0'
    char s3[] = {'h','i','\0'}; // 手动加结束符

    printf("%s\n", s1);
    printf("%s\n", s2);
    printf("%s\n", s3);

    return 0;
}

   ```
2. 获取长度
```c
int arr[] = {1, 2, 3, 4, 5};
int len = sizeof(arr) / sizeof(arr[0]);

printf("数组长度 = %d\n", len);

   ```
3. 字符串的遍历
```c
char str[] = "hello";

for(int i = 0; str[i] != '\0'; i++){
    printf("%c ", str[i]);
}

```
```c
char str[] = "hello";
int len = sizeof(str)/sizeof(char);
for(int i = 0; i<len ; i++){
    printf("%c ", str[i]);
}
```
### 二维数组
```c
int a[3][4];  // 3 行 4 列

int b[2][3] = {
    {1,2,3},
    {4,5,6}
};

for(int i = 0; i < 2; i++){
    for(int j = 0; j < 3; j++){
        printf("%d ", b[i][j]);
    }
    printf("\n");
}

```
