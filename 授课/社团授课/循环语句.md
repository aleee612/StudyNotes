## 2. 课程内容

### 循环的基本概念

- 引入问题：
让学员思考，如果要打印 10 次 Hello, World!，有什么办法？  
（可能的答案：手动写 10 行 printf()，但这显然不高效。）

- 引出循环的概念：
循环（Loop）：重复执行某一段代码，直到满足特定的条件。

- 分类：
while（条件控制循环）
do-while（至少执行一次的循环）
for（计数控制循环）

---
### while 循环
#### 语法：
```cpp
while (条件) {  
// 循环体  
}
```
#### 示例：
```cpp
#include <stdio.h>  
int main() {  
int count = 0;  
while (count < 10) {  
printf("Hello, World!\n");  
count++; // 记得更新变量，否则会陷入死循环  
}  
return 0;  
}
```
#### 重点讲解：
- while 适用于条件满足时继续循环的场景。
- 需要确保循环变量在适当时修改，否则会导致死循环。

---
### do-while 循环
- 语法：
```cpp
do {  
// 代码块  
} while (条件);
```
- 示例：
```cpp
#include <stdio.h>  
int main() {  
int num;  
do {  
printf("请输入一个正数: ");  
scanf("%d", &num);  
} while (num <= 0); // 只要输入非正数，就会要求重新输入  
printf("你输入的是 %d\n", num);  
return 0;  
}
```
- 重点讲解：
	do-while 至少执行一次，哪怕条件最初就是 false。
	适用于用户输入验证等场景。

---
### for 循环
- 语法：
```cpp
for (初始化; 条件; 递增/递减) {  
// 代码块  
}
```
- 示例：
```cpp
#include <stdio.h>  
int main() {  
for (int i = 0; i < 10; i++) {  
printf("Hello, World! %d\n", i);  
}  
return 0;  
}
```
- 重点讲解：
	for 循环更适合计数循环。
	for 的三部分：
	1. 初始化：循环变量的初始值（如 int i = 0）。
	2. 条件：循环继续执行的条件（如 i < 5）。
	3. 递增/递减：每次循环后变量如何变化（如 i++）。

---
### break 和 continue 控制循环
- break 示例（用于提前跳出循环）：
```cpp
#include <stdio.h>  
int main() {  
for (int i = 0; i < 10; i++) {  
if (i == 5) {  
break; // i 等于 5 时跳出循环  
}  
printf("%d\n", i);  
}  
return 0;  
}
```
- continue 示例（跳过本次循环，进入下一次）：
```cpp
#include <stdio.h>  
int main() {  
for (int i = 0; i < 10; i++) {  
if (i == 5) {  
continue; // 跳过 i==2 时的打印  
}  
printf("%d\n", i);  
}  
return 0;  
}
```

## 3. 课堂练习
- 基础题
1. 使用 for 循环打印 1~10 的所有整数。
2. 使用 while 计算 1+2+3+...+100 的总和。
3. 用 do-while 实现用户输入 0 退出的程序。
- 进阶题：
1. 让用户输入一个整数 n，使用 for 循环打印 n 的阶乘（n!）。
2. 让用户输入一个整数 n，判断它是否为质数。

---
# 4.  课程总结
1. while：条件为 true 时执行，适用于条件控制的情况。
2. do-while：至少执行一次，适用于用户输入校验。
3. for：计数循环，适用于已知次数的循环场景。
4. break 和 continue：控制循环的执行流程。
